> [!note]- 기능설명  
> #### 1. 탭  
> - 상단에서 **사용자 / 관리자 모드**를 전환할 수 있다.  
> - 모드에 따라 표시되는 메뉴가 달라진다.  
>
> #### 2. 신규등록  
> - 새로운 메뉴를 생성할 수 있다.  
> - 클릭 시 메뉴 등록 화면으로 이동한다.  
>
> #### 3. 엑셀  
> - 현재 메뉴 목록을 **엑셀 파일로 다운로드**할 수 있다.  
> - 보고서, 검토, 백업용으로 활용된다.  
>
> #### 4. 삭제  
> - 선택된 메뉴 항목을 삭제한다.  
> - 삭제 시 복구가 불가능하므로 주의가 필요하다.  
>
> #### 5. 체크박스  
> - 다수의 메뉴 항목을 동시에 선택할 수 있다.  
> - 선택 후 삭제 같은 일괄 작업을 할 수 있다.  
>
> #### 6. 페이지 경로  
> - 각 메뉴와 연결된 실제 URL을 확인할 수 있다.  
> - 링크를 통해 해당 페이지로 직접 이동 가능하다.  

---

> [!tip]- 사용방법  
> 1. **탭**: **탭 ①** 클릭 → 사용자 모드 또는 관리자 모드로 전환된다.
> 2. **신규등록**: **신규등록 버튼 ②** 클릭 → 새 메뉴를 입력할 수 있는 화면으로 이동한다.
> 3. **엑셀**: **엑셀 버튼 ③** 클릭 → 현재 메뉴 목록을 엑셀 파일로 내려받는다.  
> 4. **삭제**: **삭제 버튼 ④** 클릭 → 선택한 항목이 목록에서 제거된다.  
> 5. **체크박스**: **체크박스 ⑤** 선택 → 여러 메뉴를 동시에 지정해 일괄 작업을 수행한다.  
> 6. **페이지 경로**: **페이지 경로 ⑥** 클릭 → 실제 연결된 페이지로 이동한다.
