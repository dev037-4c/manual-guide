> [!note]- 기능설명
> #### 1. [신규등록](학습-그룹-신규등록.md)  
> 새로운 콘텐츠 그룹을 생성할 때 사용합니다.  
> 신규등록 버튼 클릭 시 그룹 신규등록 페이지로 이동합니다.  
> 등록 시 그룹화, 콘텐츠 추가 기능을 제공합니다.  
>
> #### 2. [엑셀](엑셀.md)  
> 엑셀 버튼 클릭 시 엑셀 다운로드 팝업 창이 생성됩니다.  
> 단순 조회용, 내부 보고용, 고객사 전달용으로 다운로드할 수 있습니다.  
>
> #### 3. 삭제  
> 삭제 버튼 클릭 시 해당 그룹을 삭제합니다. (체크박스 선택 시 활성화)  
> 삭제 시 그룹 자체는 제거되지만, 그룹 내 콘텐츠는 유지됩니다.  
>
> #### 4. 체크박스  
> 그룹 목록 선택 시 사용됩니다.  
> 체크박스를 선택하면 삭제 버튼이 활성화됩니다.  
>
> #### 5. [상세](학습-그룹-상세.md)  
> ID/제목 클릭 시 그룹 상세 페이지로 이동합니다.  
> 상세 화면에서 제목, 상태, 설명, 운영권한을 수정할 수 있습니다.  
>
> #### 6. 검색  
> ID, 제목 등 원하는 항목을 검색할 수 있습니다.  

> [!tip]- 사용방법
> - **신규등록**: 콘텐츠 그룹을 새로 생성하려면 신규등록 버튼을 클릭하고 필요한 정보를 입력합니다.  
> - **엑셀**: 그룹 목록을 다운로드하려면 엑셀 버튼을 클릭하여 팝업에서 다운로드를 진행합니다.  
> - **삭제**: 삭제할 그룹을 체크박스로 선택한 후 삭제 버튼을 클릭합니다.  
> - **체크박스**: 그룹을 삭제하거나 다중 선택할 때 체크박스를 사용합니다.  
> - **상세**: 특정 그룹의 상세 정보를 확인하거나 수정하려면 ID나 제목을 클릭합니다.  
> - **검색**: 원하는 그룹을 찾으려면 검색창에 ID, 제목 등을 입력하고 조회합니다.
