### 기능 설명

1. **추가 버튼** 클릭 시 해당 내용 추가
2. **저장 버튼** 클릭 시 해당 내용 삭제
3. **순서저장 버튼** 클릭 시 순서저장
4. **순서변경 버튼** 클릭 시 순서변경 팝업  창 생성
5. WBI 콘텐츠 Excel 일괄등록 **버튼 클릭** 시 팝업 창 생성
6. **저장 버튼** 클릭 시 순서변경 저장
7. **파일선택 버튼** 클릭 시 파일창 생성
8. **템플릿 다운로드** 클릭 시 템플릿 다운로드
