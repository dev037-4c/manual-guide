## 기능 설명  

> [!note]- #### 1. [신규등록](분류-신규등록.md)  
> - 새로운 분류를 개별 등록할 수 있는 기능
> - 클릭 시 등록 입력창이 열리며, 상위/하위 구조를 선택
> - 분류명, 코드 입력 후 저장하면 목록에 추가

> [!note]- #### 2. [엑셀](엑셀.md)  
> - 현재 조회된 목록을 엑셀 파일로 다운로드
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용

> [!note]- #### 3. 체크박스  
> - 목록의 분류 항목을 선택
> - 다중 작업(삭제 등) 시 편리하게 사용

> [!note]- #### 4. [삭제](분류-삭제.md)  
> - 선택된 분류를 삭제
> - 하위 분류 또는 연결된 데이터가 있으면 삭제가 제한될 수 있음

> [!note]- #### 5. 분류명 클릭  
> - 분류명을 클릭하면 해당 분류의 **상세/수정 화면**으로 이동
> - 상세 화면에서 내용 수정, 상태 변경, 첨부파일 관리가 가능 

> [!note]- #### 6. 수정 아이콘  
> - 해당 분류의 정보를 수정 
> - 분류명, 코드, 순서 등을 변경할 수 있으며, 팝업 또는 인라인 입력창이 열림

---

> [!tip]  
> ## 사용방법  
> - **신규 분류 등록**: `[신규등록]` 버튼 클릭 → 분류명 및 코드 입력 → 저장  
> - **분류 목록 다운로드**: `[엑셀]` 버튼 클릭 → 분류 데이터 다운로드  
> - **분류 삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **분류 수정**: 톱니바퀴 아이콘 클릭 → 수정 후 저장  
> - **다중 작업**: `[전체 선택]` 체크 후 일괄 삭제 또는 관리 가능
