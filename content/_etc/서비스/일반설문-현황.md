## 기능 설명  

> [!note]- #### 1. [엑셀](엑셀.md)  
> - 현재 조회된 설문 결과 목록을 엑셀 파일로 다운로드
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용 

> [!note]- #### 2. 미리보기  
> - 설문 응답자 화면을 **실제와 동일하게 미리보기** 
> - 문항 구성, UI/UX 확인에 유용
> - 클릭 시 팝업 또는 새 창에서 설문지 확인

> [!note]- #### 3. 검색  
> - 특정 설문 결과를 빠르게 찾을 수 있는 기능 
> - 검색 기준: 설문 제목, ID 등  
> - 검색어 입력 후 조회 시 해당 설문만 목록에 표시

> [!note]- #### 4. 상세보기  
> - 특정 설문 결과의 상세 내용을 확인
> - 클릭 시 문항별 응답 현황, 점수, 참여 인원 등 상세 데이터 조회 가능  
> - 설문 결과 분석 시 유용하게 활용 

---

> [!tip]  
> ## 사용방법  
> - **설문 결과 다운로드**: `[엑셀]` 버튼 클릭 → 설문 현황 데이터를 다운로드  
> - **설문 미리보기**: `[미리보기]` 버튼 클릭 → 실제 설문지 확인  
> - **특정 설문 검색**: 검색창에 키워드 입력 → `[조회]` 클릭  
> - **설문 상세 데이터 확인**: `[보기]` 버튼 클릭 → 문항별 결과 상세 확인
