## 기능 설명  

> [!note]- #### 1. [엑셀](엑셀.md)  
> - 현재 조회된 목록을 엑셀 파일로 다운로드
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  

> [!note]- #### 2. 학습자 이름 클릭  
> - 학습자의 이름을 클릭하면 **학습자 상세 페이지**로 이동
> - 학습자의 개별 이력, 수강 정보, 시험 결과 등을 확인

> [!note]- #### 3. 진도율  
> - 각 과정의 학습 진도율을 퍼센트(%)로 확인
> - 클릭 시 행 상태를 시각적으로 보여주며, 학습 관리에 참고

> [!note]- #### 4. 검색  
> - 학습자의 이름, ID, 과정명 등 다양한 항목으로 검색
> - 검색어 입력 후 **Enter 키** 또는 검색 아이콘 클릭 시 결과가 필터링 됨

---

> [!tip]  
> ## 사용방법  
> - **학습현황 저장**: `[엑셀]` 버튼 클릭 → 현재 조회된 학습현황 데이터를 다운로드  
> - **학습자 상세 확인**: 학습자 이름 클릭 → 상세 이력 페이지 이동  
> - **진도율 확인**: 진도율 항목에서 학습 진행 상황 확인  
> - **검색 기능 활용**: 검색창에 이름, ID, 과정명 입력 → 원하는 학습자 조회
