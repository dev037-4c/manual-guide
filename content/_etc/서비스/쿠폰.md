> [!note]- 기능 설명  
> 1. 신규등록  
> - 새로운 쿠폰을 등록할 때 사용  
> - 버튼 클릭 시 **쿠폰 신규등록 페이지**로 이동  
> - 쿠폰명, 할인조건, 사용기간 등 필수 항목을 입력하여 등록  
>  
> 2. 엑셀  
> - 현재 조회된 쿠폰 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
>  
> 3. 체크박스  
> - 개별 쿠폰을 선택  
> - 다중 선택이 가능하며, 선택 시 상단의 **[삭제] 버튼**이 활성화  
>  
> 4. 삭제  
> - 체크박스로 선택한 쿠폰을 삭제  
> - 삭제 시 쿠폰 데이터는 즉시 제거되며 복구할 수 없음  
>  
> 5. 상세 페이지 이동  
> - 쿠폰명(제목)을 클릭하면 해당 쿠폰의 **상세 페이지**로 이동  
> - 상세 화면에서 제목, 할인조건, 사용기간, 사용여부 등을 수정  
>  
> 6. 쿠폰발급  
> - 선택한 쿠폰을 **직접 발급**할 수 있음  
> - 발급 버튼 클릭 시 발급 팝업 또는 대상자 선택 창이 실행  
> - 특정 회원 또는 전체 회원 대상으로 발급 가능  
>  
> 7. 사용/청구 내역  
> - 발급된 쿠폰의 **사용 현황**을 표시  
> - 예시: `0/2 (사용률 0%)`  
>   - 앞 숫자 = 사용된 쿠폰 수  
>   - 뒷 숫자 = 총 발급된 쿠폰 수  
>   - 퍼센트 = 발급 대비 사용 비율  
> - 클릭 시 상세 내역(사용자, 사용일자 등) 확인 가능  
>  
> 8. 검색  
> - 원하는 쿠폰을 빠르게 찾을 수 있음  
> - **제목, 등록일, 등록자 등 조건**으로 검색  

---

> [!tip]- 사용방법  
> - **신규 쿠폰 등록**: `[신규등록]` 버튼 클릭 → 쿠폰 정보 입력 → 저장  
> - **쿠폰 목록 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **쿠폰 삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **쿠폰 상세 조회/수정**: 쿠폰명 클릭 → 상세 페이지 이동  
> - **쿠폰 발급**: `[쿠폰발급]` 버튼 클릭 → 회원/그룹 선택 후 발급  
> - **쿠폰 사용내역 확인**: 사용/청구 내역 영역 클릭 → 상세 사용 현황 확인  
> - **쿠폰 검색**: 검색창에 키워드 입력 → `[조회]` 클릭
