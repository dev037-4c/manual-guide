> [!note]- 기능 설명  
> 1. [신규등록](포인트-신규등록.md)  
> - 새로운 포인트 규칙을 생성할 때 사용  
> - 버튼 클릭 시 **포인트 등록 페이지**로 이동  
> - 포인트명, 적용수치, 사용 여부 등을 설정하여 등록  
>   
> 2. [엑셀](엑셀.md)  
> - 현재 조회된 포인트 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
>   
> 3. 체크박스  
> - 개별 포인트 규칙을 선택  
> - 체크박스 선택 후 삭제 버튼을 클릭하면 선택된 항목이 삭제  
>   
> 4. 삭제  
> - 선택한 포인트를 목록에서 삭제  
> - 삭제 전 확인 메시지가 표시되며, 삭제 후 복구할 수 없음  
>   
> 5. 상세  
> - 포인트 규칙 제목을 클릭하면 해당 포인트의 **상세/수정 화면**으로 이동  
> - 상세 화면에서 적용수치, 사용 여부, 설명 등을 확인하고 수정  
>   
> 6. 검색  
> - 원하는 포인트 규칙을 빠르게 찾을 수 있음  
> - **제목, 등록자, 기간 등 조건**으로 검색할 수 있음  

---

> [!tip]- 사용방법  
> - **신규 포인트 규칙 등록**: `[신규등록]` 버튼 클릭 → 포인트 정보 입력 → 저장  
> - **포인트 목록 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **포인트 삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **포인트 상세 확인/수정**: 제목 클릭 → 상세/수정 페이지 이동  
> - **포인트 검색**: 검색창에 조건 입력 → `[조회]` 클릭
