## 기능 설명  

> [!note]- #### 1. [엑셀](엑셀.md)  
> - 현재 조회된 게시글 목록을 엑셀 파일로 다운로드
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용

> [!note]- #### 2. 체크박스  
> - 개별 게시글을 선택
> - 다중 선택이 가능하며, 선택 시 승인, 반려, 사용, 미사용, 삭제 등 상단 메뉴와 함께 사용

> [!note]- #### 3. 승인  
> - 선택된 게시글을 **승인 상태**로 변경
> - 승인된 게시글은 학습자 화면에서 노출

> [!note]- #### 4. 반려  
> - 선택된 게시글을 **반려 상태**로 변경
> - 반려 시 학습자 화면에 노출되지 않으며, 반려 사유 입력이 가능(시스템 설정에 따라 다름).  

> [!note]- #### 5. 사용  
> - 선택된 게시글을 **사용 상태**로 전환  
> - 사용 상태일 경우 LMS 화면에 정상적으로 노출 

> [!note]- #### 6. 미사용  
> - 선택된 게시글을 **미사용 상태**로 전환
> - 미사용 상태일 경우 LMS 화면에서 노출 안됨

> [!note]- #### 7. 삭제  
> - 선택된 게시글을 삭제
> - 삭제 전 확인 메시지가 표시되며, 선택된 항목만 삭제

> [!note]- #### 8. 게시글 제목  
> - 게시글 제목을 클릭하면 **상세 페이지**로 이동
> - 상세 화면에서 내용 수정, 상태 변경, 첨부파일 관리가 가능

> [!note]- #### 9. 검색  
> - 검색창에 키워드를 입력해 게시글을 조회
> - 제목, 등록자, 상태 등 조건별 검색이 가능

---

> [!tip]  
> ## 사용방법  
> - **게시글 목록 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **게시글 승인**: 체크박스 선택 후 `[승인]` 버튼 클릭  
> - **게시글 반려**: 체크박스 선택 후 `[반려]` 버튼 클릭  
> - **게시글 사용 처리**: 체크박스 선택 후 `[사용]` 버튼 클릭  
> - **게시글 미사용 처리**: 체크박스 선택 후 `[미사용]` 버튼 클릭  
> - **게시글 삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **상세 확인/수정**: 게시글 제목 클릭 → 상세 페이지 이동  
> - **게시글 검색**: 검색창에 키워드 입력 → `[조회]` 클릭
