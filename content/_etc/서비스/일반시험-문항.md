> [!note]- 기능 설명  
> 1. 신규등록  
> - 새로운 문항을 개별 등록할 때 사용  
> - `[신규등록]` 버튼 클릭 시 문항 등록 페이지로 이동  
> - 문제 유형, 보기, 정답, 난이도 등을 입력하여 등록  
>  
> 2. 엑셀  
> - 현재 조회된 문항 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
>  
> 3. 일괄등록  
> - 여러 문항을 엑셀 양식으로 한 번에 등록  
> - 대량 문항 관리 시 효율적  
> - 지정된 템플릿 파일에 문항을 입력 후 업로드  
>  
> 4. 제목 클릭  
> - 문항 제목 클릭 시 **상세 페이지**로 이동  
> - 문항 내용, 보기, 정답, 난이도 등을 확인 및 수정 가능  
> - 수정 후 저장 시 해당 문항이 시험지에 반영  
>  
> 5. 미리보기  
> - 문항을 응시자 화면과 동일하게 확인  
> - 객관식, 단답형, 서술형 등 유형별 표시 확인 가능  
> - 정답 및 문항 배치 확인에도 유용  
>  
> 6. 검색  
> - 문항을 ID, 제목 등으로 검색  
> - 원하는 문항만 빠르게 조회 가능  

---

> [!tip]- 사용방법  
> - **신규 문항 등록**: `[신규등록]` 버튼 클릭 → 문제 유형 및 내용 입력 → 저장  
> - **문항 목록 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **문항 일괄등록**: `[일괄등록]` 버튼 클릭 → 엑셀 템플릿 업로드  
> - **문항 상세 수정**: 문항 제목 클릭 → 상세 페이지에서 수정  
> - **문항 미리보기**: `[미리보기]` 버튼 클릭 → 응시자 화면과 동일하게 확인  
> - **문항 검색**: 검색창에 ID, 제목 입력 → `[조회]` 클릭
