> [!note]- 기능 설명  
> 1. 신규등록  
> - 새로운 당첨자 발표를 등록할 때 사용  
> - `[신규등록]` 버튼 클릭 시 당첨자 발표 등록 페이지로 이동  
> - 제목, 기간, 내용 등을 입력 후 등록  
>
> 2. 엑셀  
> - 현재 조회된 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
>
> 3. 체크박스  
> - 당첨자 발표 항목을 선택  
> - 다중 선택이 가능하며, 선택 시 **[삭제]** 버튼 활성화  
>
> 4. 삭제  
> - 선택한 당첨자 발표를 목록에서 삭제  
> - 삭제 시 데이터가 시스템에서 제거되며 복구 불가능  
>
> 5. 상세  
> - 제목을 클릭하면 당첨자 발표 **상세 페이지**로 이동  
> - 상세 화면에서 내용 확인 및 수정이 가능  
>
> 6. URL복사  
> - 선택한 당첨자 발표의 **공유용 URL을 복사**  
> - 복사한 URL은 외부 공지, 안내 메일, 게시판 등에 활용  
>
> 7. 검색  
> - 원하는 당첨자 발표를 빠르게 찾을 수 있음  
> - **제목, 등록일, 등록자 등 조건**으로 검색  

---

> [!tip]- 사용방법  
> - **신규 등록**: `[신규등록]` 버튼 클릭 → 제목/기간/내용 입력 → 저장  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **상세 확인/수정**: 제목 클릭 → 상세 페이지 이동  
> - **URL 공유**: `[URL복사]` 버튼 클릭 → 외부 공유  
> - **검색**: 검색창에 조건 입력 → `[조회]` 클릭
