> [!note]- 기능 설명  
> 1. 신규등록  
> - 새로운 이벤트를 생성할 때 사용  
> - `[신규등록]` 버튼 클릭 시 이벤트 등록 페이지로 이동  
> - 이벤트명, 기간, 노출 여부, 이미지, 조건 등을 입력하여 등록  
>  
> 2. 엑셀  
> - 현재 조회된 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
>  
> 3. 체크박스  
> - 이벤트 항목을 선택  
> - 다중 선택이 가능하며, 선택 시 **[삭제] 버튼**이 활성화  
>  
> 4. 삭제  
> - 선택한 이벤트를 목록에서 삭제  
> - 삭제 시 데이터가 시스템에서 제거되며 복구할 수 없음  
>  
> 5. 상세  
> - 이벤트 제목 클릭 시 **상세 페이지**로 이동  
> - 상세 화면에서 이벤트 기본정보, 조건, 참여 현황 등을 확인 및 수정  
>  
> 6. 미리보기  
> - 이벤트 화면을 실제 사용자 화면과 동일하게 **미리보기** 할 수 있음  
> - 등록 완료 전에 화면 노출 상태를 확인하여 운영 효율성을 높일 수 있음  
>  
> 7. 검색  
> - 원하는 이벤트를 빠르게 찾을 수 있음  
> - **제목, 등록일, 등록자 등 조건**으로 검색  

---

> [!tip]- 사용방법  
> - **신규 이벤트 등록**: `[신규등록]` 버튼 클릭 → 이벤트 정보 입력 → 저장  
> - **이벤트 목록 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **상세 확인/수정**: 이벤트 제목 클릭 → 상세 페이지 이동  
> - **미리보기**: `[미리보기]` 버튼 클릭 → 실제 노출 화면 확인  
> - **검색**: 검색창에 조건 입력 → `[조회]` 클릭
