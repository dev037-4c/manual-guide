> [!note]- 기능 설명  
> 1. 정렬 기준 선택  
> - `조회 순위`, `학습시간 순위`, `학습자 순위` 중 하나를 선택하여 원하는 기준의 랭킹 확인  
> - 클릭 시 해당 기준에 맞는 순위표로 자동 갱신  
>
> 2. 엑셀  
> - 현재 조회된 랭킹 데이터를 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  

---

> [!tip]- 사용방법  
> - **랭킹 확인**: 상단에서 정렬 기준 `[조회순위] / [학습시간순위] / [학습자순위]` 중 선택  
> - **엑셀 저장**: `[엑셀]` 버튼 클릭 → 현재 화면의 랭킹 데이터를 다운로드
