> [!note]- 기능설명  
> #### 1. [목록](일반시험-문항.md)  
> - 문항 관리 목록 화면으로 돌아갈 수 있다.  
> - 작성 중인 내용은 저장되지 않으므로 주의가 필요하다.  
>
> #### 2. 저장  
> - 입력한 문항 정보를 저장한다.  
> - 필수 입력 항목이 누락될 경우 저장되지 않는다.  
>
> #### 3. 삭제  
> - 선택된 문항을 삭제한다.  
> - 삭제 시 복구가 불가능하므로 신중히 사용해야 한다.  
>
> #### 4. 분류 선택  
> - 문항이 속할 학습 과정의 분류를 지정한다.  
> - 문항 관리의 체계적 분류를 위해 사용된다.  
>
> #### 5. 출제 그룹 선택  
> - 문항이 포함될 시험 그룹을 설정한다.  
> - 시험지 편성 시 그룹별 문항 활용이 가능하다.  
>
> #### 6. 파일 선택(문항 이미지)  
> - 문항에 삽입할 이미지를 업로드할 수 있다.  
> - 시각적 자료를 통해 학습자의 이해를 돕는다.  
>
> #### 7. 파일 선택(해설)  
> - 문항 해설을 보조 자료(파일)로 업로드할 수 있다.  
> - PDF, 이미지 등을 첨부하여 학습자에게 제공 가능하다.  

---

> [!tip]- 사용방법  
> - **목록**: 상단의 **목록 버튼①** 클릭 → 문항 관리 목록 화면으로 이동한다.  
> - **저장**: 필수 항목을 입력 후 **저장 버튼②** 클릭 → 현재 문항이 저장된다.  
> - **삭제**: 불필요한 문항은 **삭제 버튼③** 클릭 → 확인 창에서 ‘확인’을 눌러 삭제한다.  
> - **분류 선택**: 문항의 소속 과정을 지정하려면 **분류 선택 버튼④** 클릭 → 원하는 분류를 선택한다.  
> - **출제 그룹 선택**: 시험 그룹을 지정하려면 **출제 그룹 선택 버튼⑤** 클릭 → 지정된 그룹에 문항이 포함된다.  
> - **파일 선택(문항 이미지)**: 문항에 이미지를 추가하려면 **파일선택 버튼⑥** 클릭 → 이미지 파일 업로드.  
> - **파일 선택(해설)**: 문항 해설 자료를 추가하려면 **파일선택 버튼⑦** 클릭 → PDF, 이미지 등 업로드.
