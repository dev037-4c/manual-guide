# 9. 콘텐츠

## [[시험-출제그룹]]

---



---

> **[콘텐츠] → [시험] → [시험지]** 을 클릭하여 이동한 페이지

---

### 기능 설명

1. **신규등록 버튼** 클릭 시 시험지 신규등록 페이지로 이동  
2. **엑셀 버튼** 클릭 시 엑셀 다운로드 팝업 창 생성  
3. **다운로드 버튼** 클릭 시 설정한 사유에 맞는 엑셀 파일 다운로드  
4. 원하는 검색어 입력 시 필터링 된 목록 표시
5. **ID / 제목** 클릭 시 해당 시험지 미리보기 팝업 창 생성  
6. **미리보기 버튼** 클릭 시 해당 시험지 미리보기 팝업 창 생성
7. **문항** 클릭 시 해당 시험지 문항 목록 페이지로 이동
