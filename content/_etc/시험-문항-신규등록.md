## 신규등록

새로운 학습 콘텐츠를 등록하거나 기존 자료를 수정할 수 있는 페이지입니다.  
콘텐츠 유형, 파일, 자막, 재생시간, 플레이어 설정 등 기본 정보를 입력하고 저장할 수 있습니다.

***

![[시험-문항-신규등록.png]]

***

> **[콘텐츠] → [시험] → [문항] → [신규등록] 클릭하여 이동한 페이지

***

## 기능 설명

#### 1. 목록

- 클릭 시 문항 목록 화면으로 이동

---

#### 2. 저장

- 입력된 문항 정보를 저장
- 필수 항목 미입력 시 저장 불가

---

#### 3. [분류](분류.md)

- 선택 버튼 클릭 시 분류선택 팝업 창 생성
- 하위단계 분류를 선택해야 등록 가능

---

#### 4. [과정 템플릿](과정템플릿.md)

- 선택 버튼 클릭 시 과정템플릿 선택 팝업 창 생성

---

#### 5. [출제그룹](시험-문항-신규등록_출제그룹.md)

- 선택 버튼 클릭 시 출제그룹 선택 팝업 창 생성
- 문항을 특정 출제그룹에 등록 가능

---

#### 6. 파일 선택

- 파일 선택 버튼 클릭 시 파일 선택 창 생성
- 폴더에서 업로드할 파일 선택 시 추가 가능

---

#### 7. 보기

- 문항의 보기를 입력 및 관리
- 보기 개수 설정 가능 (예: 4지선다, 5지선다)
- 각 보기에 텍스트 입력, 이미지 업로드, 파일 첨부 가능
- 정답 지정 및 보기별 관리 가능

---

## 사용방법

- 문항 목록 화면으로 이동하려면 **목록 버튼**을 클릭합니다.
- 입력한 문항 정보를 저장하려면 **저장 버튼**을 클릭하며, 필수 항목이 누락되면 저장되지 않습니다.
- 분류를 지정하려면 **분류 선택 버튼**을 클릭하고 팝업에서 하위 분류를 선택합니다.
- 과정 템플릿을 지정하려면 **과정 템플릿 버튼**을 클릭하고 팝업에서 원하는 템플릿을 선택합니다.
- 문항을 특정 출제그룹에 등록하려면 **출제그룹 버튼**을 클릭하고 팝업에서 그룹을 선택합니다.
- 문항에 필요한 파일을 첨부하려면 **파일 선택 버튼**을 클릭하고 로컬 폴더에서 업로드할 파일을 지정합니다.
- 문항 보기를 입력·관리하려면 **보기 설정 기능**을 사용하여 보기 개수를 정하고, 각 보기에 텍스트 입력·이미지 업로드·파일 첨부·정답 지정 등을 설정합니다.
