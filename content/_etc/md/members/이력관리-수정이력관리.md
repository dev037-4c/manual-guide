> [!note]- 기능설명
> 
> #### 1. 검색 (기간)
> 
> - 조회 기간을 설정하여 수정 이력을 검색할 수 있다.
>     
> - 특정 기간 동안의 변경 내역을 확인할 수 있다.
>     
> 
> #### 2. [엑셀](excel.md)
> 
> - 수정 이력 내역을 엑셀 파일로 내려받을 수 있다.
>     
> - 자료 관리 및 보고용으로 활용된다.
>     
> 
> #### 3. [상세정보](이력관리-상세.md)
> 
> - 수정 작업을 수행한 사용자의 이름과 ID가 표시된다.
>     
> - 클릭 시 해당 사용자의 상세정보 페이지로 이동할 수 있다.
>     
> 
> #### 4. 검색
> 
> - 이름, ID, 메뉴, 항목 등 조건을 입력해 수정 이력을 검색할 수 있다.
>     
> - 원하는 내역을 빠르게 조회할 수 있다.
>     
> 
> #### 5. [DATA 버튼](수정이력관리-DATA버튼.md)
> 
> - 해당 수정 내역의 상세 데이터를 확인할 수 있다.
>     
> - 변경된 값과 항목별 상세 정보를 조회 가능하다.
>     

---

> [!tip]- 사용방법
> 
> - **검색 (기간)**: 상단의 **기간 설정 ①** 후 검색 버튼을 클릭 → 해당 기간의 수정 이력을 조회한다.
>     
> - **엑셀 다운로드**: 상단의 **엑셀 버튼 ②**을 클릭 → 수정 이력 데이터를 저장한다.
>     
> - **사용자 정보 확인**: 목록의 **이름/ID ③**을 클릭 → 해당 사용자의 상세 페이지로 이동한다.
>     
> - **조건 검색**: 상단의 **검색창 ④**에 조건을 입력 → 특정 수정 내역을 빠르게 조회한다.
>     
> - **DATA 확인**: 목록의 **DATA 버튼 ⑤**을 클릭 → 수정된 상세 데이터를 확인한다.
>
