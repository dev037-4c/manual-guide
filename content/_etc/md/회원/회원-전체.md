> [!note]- 기능설명
> 
> #### 1. [엑셀](엑셀.md)
> 
> - 전체 회원 목록을 엑셀 파일로 다운로드 받을 수 있다.
>     
> - 외부 보고서 작성이나 데이터 백업에 활용할 수 있다.
>     
> 
> #### 2. SMS
> 
> - 선택한 회원에게 문자 메시지를 발송한다.
>     
> - 공지사항이나 주요 알림을 전달할 때 사용된다.
>     
> 
> #### 3. 이메일
> 
> - 선택한 회원에게 이메일을 발송한다.
>     
> - 대량 메일 발송이나 안내 공지에 활용 가능하다.
>     
> 
> #### 4. 승인
> 
> - 가입 대기 중인 회원을 승인 처리한다.
>     
> - 승인된 회원만 정상적으로 서비스를 이용할 수 있다.
>     
> 
> #### 5. 휴면처리
> 
> - 장기간 미 접속 회원을 휴면 상태로 전환한다.
>     
> - 보안 강화 및 데이터 관리 목적에 활용된다.
>     
> 
> #### 6. 탈퇴
> 
> - 선택된 회원을 시스템에서 탈퇴 처리한다.
>     
> - 탈퇴된 회원의 정보는 더 이상 조회할 수 없다.
>     
> 
> #### 7.  로그인 잠금 해제
> 
> - 로그인 실패로 인해 잠긴 계정을 해제한다.
>     
> - 잠금 해제 후 회원은 정상적으로 로그인할 수 있다.
>     
> 
> #### 8. 체크박스
> 
> - 개별 또는 전체 회원을 선택할 수 있다.
>     
> - 선택한 회원에게 일괄적으로 문자, 이메일, 탈퇴 등의 처리를 한다.
>  
> #### 9. 상태 표시 아이콘
> - 회원의 접속 상태나 활동 여부를 시각적으 표시한다.
> - 활성 회원과 비활성 회원을 구분하는데 도움이 된다.
> 
> #### 10. 회원 목록
> - 등록된 전체 회원의 이름, ID, 연락처, 이메일, 상태, 가입일 등 을 표시된다.
> - 컬럼별 정렬과 검색이 가능하다.
> 
> #### 11. [메신저](전체-메신저.md)
> - 각 회원의 번호로 직접 문자를 전송할 수 있다.
> - 클릭 시 문자 팝업 창이 생성된다.
> 
> #### 12. [이메일](전체-이메일.md)
> - 각 회원의 이메일 주소로 직접 메일을 발송할 수 있다.
> - 클릭 시 기본 메일 팝업 창이 생성된다.
> 
> #### 13. 검색
> - 이름, ID 등으로 조건 검색을 할 수 있다.
> - 원하는 회원을 빠르게 찾을 수 있다.

---

> [!tip]- 사용방법
> 
> - **엑셀 다운로드**: 상단의 **엑셀 버튼 ①**을 클릭 → 전체 회원 목록을 파일로 저장한다.
> - **SMS**: **SMS 버튼②**을 클릭 → 문자 작성 팝업 생성 → 메시지를 입력하고 전송 버튼 클릭 → 선택된 회원에게 문자가 발송된다.
> - **이메일**: **이메일 버튼③**을 클릭 → 메일 작성 팝업 생성 → 제목과 내용을 입력하고 보내기 버튼 클릭 → 메일을 발송한다.
> - **승인**: **승인 버튼④**을 클릭 → 가입이 승인되어 정상적인 서비스 이용이 가능해진다.
> - **휴면처리**: 회원을 선택 후 **휴면처리 버튼 ⑤**를 클릭 → 계정을 휴면 상태로 전환한다.
> - **탈퇴**: 탈퇴할 회원을 체크박스로 선택 → **탈퇴 버튼⑥**을 클릭 → 회원 정보가 삭제되며 목록에서 사라진다.
> - **로그인 잠금 해제**: 잠긴 계정을 선택 후 **로그인 잠금 해제 버튼 ⑦**를 클릭 → 정상 로그인 가능 상태로 만든다.
> - **체크박스**: 여러 회원을 동시에 작업할 경우 **선택박스 ⑧**을 활용한다.
> - **상태 표시 아이콘**: 회원 목록에서 **상태 아이콘⑨**을 확인하면 접속 여부를 즉시 파악할 수 있으며, 초록색은 활성 회원, 회색은 비활성 회원을 의미한다.
> - **이름/ID**: 화면 중앙의 **이름/ID⑩**에서 이름, ID, 연락처, 이메일, 상태, 가입일 등을 확인 → 각 컬럼 제목을 클릭 → 상단 검색창에 키워드를 입력 → 특정 회원을 조회할 수 있다.
> - **문자 아이콘**:**문자 아이콘⑪**을 클릭 → 문자 작성 팝업 창 생성 → 메시지를 입력하고 ‘전송’ 버튼을 눌러 발송한다.
> - **이메일 아이콘**:**이메일 아이콘⑫**을 클릭 → 기본 메일 작성 팝업 창이 생성 → 제목과 내용을 작성한 후 ‘보내기’를 눌러 메일을 발송한다.
> - **검색**: 상단의 **검색창 ⑬**에 조건 입력 → 조회 버튼을 눌러 원하는 회원을 찾는다.
