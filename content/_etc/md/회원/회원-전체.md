> [!note]- 기능설명
> 
> #### 1. [엑셀](엑셀.md)
> 
> - 전체 회원 목록을 엑셀 파일로 내려받을 수 있다.
>     
> - 외부 보고서 작성이나 데이터 백업에 활용할 수 있다.
>     
> 
> #### 2. 휴면처리
> 
> - 장기간 미접속한 회원 계정을 휴면 상태로 전환할 수 있다.
>     
> - 휴면 전환 시 해당 회원은 로그인이 제한된다.
>     
> 
> #### 3. 탈퇴
> 
> - 선택한 회원 계정을 탈퇴 처리할 수 있다.
>     
> - 탈퇴된 회원은 더 이상 서비스에 접근할 수 없다.
>     
> 
> #### 4. 로그인 잠금 해제
> 
> - 비밀번호 오류 등으로 잠긴 계정을 해제할 수 있다.
>     
> - 로그인 제한을 해제하여 정상 접속이 가능하도록 한다.
>     
> 
> #### 5. 검색
> 
> - 이름, ID, 이메일, 상태 등 조건으로 회원을 검색할 수 있다.
>     
> - 원하는 조건의 회원을 빠르게 찾을 수 있다.
>     
> 
> #### 6. 체크박스
> 
> - 다수의 회원을 동시에 선택할 수 있다.
>     
> - 선택 후 일괄 휴면처리, 탈퇴 등의 작업을 적용할 수 있다.
>     
> 
> #### 7.  아이콘
> 
> - 회원의 현재 를 색상 아이콘으로 표시한다.
>     
> - 시각적으로 상태를 쉽게 파악할 수 있다.
>     
> 
> #### 8. [상세](강사-상세.md)
> 
> - 회원의 이름, ID, 이메일, 연락처, 가입일, 최근접속일 등을 확인할 수 있다.
>     
> - 클릭 시 해당 회원의 상세 정보 화면으로 이동할 수 있다.
>  
> #### 9. 상태 표시 아이콘
> - 회원의 접속 상태나 활동 여부를 시각적으 표시한다.
> - 활성 회원과 비활성 회원을 구분하는데 도움이 된다.
> 
> #### 10. 회원 목록
> - 등록된 전체 회원의 이름, ID, 연락처, 이메일, 상태, 가입일 등 을 표시된다.
> - 컬럼별 정렬과 검색이 가능하다.
> 
> #### 11. 문자 아이콘
> - 각 회원의 번호로 직접 문자를 전송할 수 있다.
> - 클릭 시 문자 팝업 창이 생성된다.
> 
> #### 12. 이메일 아이콘
> - 각 회원의 이메일 주소로 직접 메일을 발송할 수 있다.
> - 클릭 시 기본 메일 팝업 창이 생성된다.

---

> [!tip]- 사용방법
> 
> - **엑셀 다운로드**: 상단의 **엑셀 버튼 ①**을 클릭 → 전체 회원 목록을 파일로 저장한다.
>     
> - **휴면처리**: 회원을 선택 후 **휴면처리 버튼 ②**를 클릭 → 계정을 휴면 상태로 전환한다.
>     
> - **탈퇴**: 불필요한 계정을 선택 후 **탈퇴 버튼 ③**을 클릭 → 회원을 삭제 처리한다.
>     
> - **로그인 잠금 해제**: 잠긴 계정을 선택 후 **로그인 잠금 해제 버튼 ④**를 클릭 → 정상 로그인 가능 상태로 만든다.
>     
> - **검색**: 상단의 **검색창 ⑤**에 조건 입력 → 조회 버튼을 눌러 원하는 회원을 찾는다.
>     
> - **체크박스**: 여러 회원을 동시에 작업할 경우 **선택박스 ⑥**을 활용한다.
>     
> - **상태 확인**: 회원의 상태는 좌측의 **상태 아이콘 ⑦**으로 확인한다.
>     
> - **상세**: 목록에서 이름이나 ID를 클릭하면 **회원 정보 ⑧** 화면으로 이동한다.
>
> - **상태 표시 아이콘**: 회원 목록에서 **상태 아이콘⑨**을 확인하면 접속 여부를 즉시 파악할 수 있으며, 초록색은 활성 회원, 회색은 비활성 회원을 의미한다.
> 
> - **회원 목록**: 화면 중앙의 **회원 목록⑩**에서 이름, ID, 연락처, 이메일, 상태, 가입일 등을 확인 → 각 컬럼 제목을 클릭 → 상단 검색창에 키워드를 입력 → 특정 회원을 조회할 수 있다.
> 
> - **문자 아이콘**: 문자 전송이 필요한 회원의 행에서 **문자 아이콘③**을 클릭 → 문자 작성 팝업 창 생성 → 메시지를 입력하고 ‘전송’ 버튼을 눌러 발송한다.
> 
> - **이메일 아이콘**: 메일을 보낼 회원의 행에서 **이메일 아이콘④**을 클릭 → 기본 메일 작성 팝업 창이 생성 → 제목과 내용을 작성한 후 ‘보내기’를 눌러 메일을 발송한다.
