> [!note]- 기능설명
> 
> #### 1. 목록
> 
> 학습자 목록으로 이동할 수 있다.  
> 이미 등록된 학습자 정보를 다시 확인하거나 수정할 때 활용된다.
> 
> #### 2. 저장
> 
> 학습자 정보를 저장할 수 있다.  
> 신규 등록뿐만 아니라 수정된 정보도 반영된다.
> 
> #### 3. 중복확인
> 
> 입력한 ID가 기존 학습자와 중복되는지 확인할 수 있다.  
> 중복 시 새로운 ID로 다시 등록해야 한다.
> 
> #### 4. 소속 선택
> 
> 학습자의 소속을 지정할 수 있다.  
> 조직별 학습자 관리와 통계를 위해 필요하다.
> 
> #### 5. 주소찾기
> 
> 우편번호와 주소를 검색하여 자동 입력할 수 있다.  
> 정확한 주소 입력을 통해 연락 및 문서 발송 시 오류를 줄인다.

---

> [!tip]- 사용방법
> 
> - **목록 이동**: 상단의 **목록 버튼 ①**을 클릭 → 학습자 목록 화면으로 이동한다.
>     
> - **저장**: 학습자 정보를 입력 후 **저장 버튼 ②**을 클릭 → 신규 또는 수정 내용을 저장한다.
>     
> - **중복확인**: ID 입력 후 **중복확인 버튼 ③**을 클릭 → ID 중복 여부를 확인한다.
>     
> - **소속 선택**: 소속 입력란에서 **소속 선택 버튼 ④**을 클릭 → 소속을 지정한다.
>     
> - **주소찾기**: 하단의 **주소찾기 버튼 ⑤**를 클릭 → 우편번호 검색 후 주소를 자동 입력한다.
>
