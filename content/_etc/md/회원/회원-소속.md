> [!note]- 기능설명
>
> #### 1. 목록  
> - 소속 목록 화면으로 돌아간다.  
> - 수정 중이던 내용을 취소하고 전체 목록을 확인할 수 있다.  
>
> #### 2. 저장  
> - 입력하거나 수정한 소속 정보를 저장한다.  
> - 변경된 내용이 시스템에 반영된다.  
>
> #### 3. 삭제  
> - 선택한 소속 정보를 삭제한다.  
> - 삭제된 소속은 복구되지 않는다.  
>
> #### 4. 순서  
> - 소속의 노출 순서를 조정한다.  
> - 상위/하위 소속 구조에서 순서를 변경할 수 있다.  
>
> #### 5. 주소찾기  
> - 우편번호를 검색하여 자동으로 주소를 입력한다.  
> - 상세 주소는 별도로 추가 입력할 수 있다.  
>
> #### 6. 지도 확인  
> - 입력된 주소를 지도에서 확인할 수 있다.  
> - 위치 정보를 시각적으로 검증할 수 있다.  

---

> [!tip]- 사용방법
>
> - **목록 이동**: 상단의 **목록 버튼 ①** 클릭 → 전체 소속 목록 화면으로 돌아간다.  
> - **소속 저장**: 수정 후 **저장 버튼 ②** 클릭 → 변경된 소속 정보를 반영한다.  
> - **소속 삭제**: 불필요한 경우 **삭제 버튼 ③** 클릭 → 해당 소속을 제거한다.  
> - **순서 변경**: **순서 버튼 ④** 클릭 → 소속 노출 순서를 조정한다.  
> - **주소 입력**: 우편번호 입력 후 **주소찾기 버튼 ⑤** 클릭 → 자동으로 주소를 불러온다.  
> - **지도 확인**: **지도 확인 버튼 ⑥** 클릭 → 입력된 주소의 위치를 지도로 검토한다.
