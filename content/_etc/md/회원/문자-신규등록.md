> [!note]- 기능 설명
> #### 1. [목록](content/_etc/md/설정/템플릿-자동알림.md)  
> 문자 템플릿 목록 화면으로 이동한다.
> 신규 작성 후 전체 현황을 다시 확인할 때 활용된다.
>
> #### 2. 저장  
> 입력한 문자 템플릿을 저장한다.
> 등록 후 즉시 시스템에 반영된다.
>
> #### 3. 삭제  
> 선택한 문자 템플릿을 삭제한다.
> 불필요한 데이터 정리에 활용된다.
>
> #### 4. 유형  
> 문자 템플릿의 유형을 지정한다.
> 예: 회원가입, 비밀번호 변경, 과정안내 등.
>
> #### 5. 사용여부  
> 등록된 문자 템플릿의 사용 상태를 설정한다.
>
> #### 6. 운영권한  
> 작성된 문자 템플릿을 운영에 반영할 수 있다.
> 전체 또는 특정 등록자 대상으로 설정된다.

---

> [!tip]- 사용 방법
> - **목록**: 상단의 **목록 버튼 ①** 클릭 → 문자 템플릿 목록 화면으로 이동한다.
> - **저장**: 작성 완료 후 **저장 버튼 ②** 클릭 → 신규 문자 템플릿이 저장된다.
> - **삭제**: 불필요한 템플릿 선택 후 **삭제 버튼 ③** 클릭 → 데이터에서 제거된다.
> - **유형**: 드롭다운에서 **유형 옵션 ④** 선택 → 문자 목적에 맞는 분류를 지정한다.
> - **사용여부**: **사용여부 옵션 ⑤** 클릭 → ‘예/아니오’로 사용 상태를 지정한다. 
> - **운영권한**: 하단의 **운영권한 옵션 ⑥** 선택 → 전체 또는 등록자별로 적용된다.
