> [!note]- 기능설명
>
> #### 1. 목록  
> - 휴면/탈퇴 회원 목록 화면으로 돌아간다.  
> - 상세보기 종료 후 전체 목록 확인 시 사용된다.  
>
> #### 2. 저장  
> - 입력/수정된 정보를 저장한다.  
> - 회원 상세 정보가 시스템에 반영된다.  
>
> #### 3. 복구  
> - 선택된 휴면/탈퇴 회원 계정을 복원한다.  
> - 정상 회원으로 전환되어 로그인 가능하다.  
>
> #### 4. 삭제  
> - 선택된 휴면/탈퇴 회원을 영구 삭제한다.  
> - 삭제된 계정은 복구할 수 없다.  
>
> #### 5. 질문하기  
> - 해당 회원과 관련된 문의를 작성할 수 있다.  
> - 관리자가 직접 확인 및 대응한다.  
>
> #### 6. 로그인 이력  
> - 회원의 로그인 내역을 확인할 수 있다.  
> - 접속 기록을 검토할 수 있다.  
>
> #### 7. 비밀번호 변경  
> - 해당 회원의 비밀번호를 재설정한다.  
> - 보안 문제나 초기화 시 활용된다.  
>
> #### 8. 소속 선택  
> - 회원의 소속을 지정하거나 변경한다.  
> - 학습자/강사/조교 등의 구분 설정에 사용된다.  
>
> #### 9. 주소찾기  
> - 우편번호 검색을 통해 주소를 자동 입력한다.  
> - 정확한 회원 주소를 등록할 수 있다.  

---

> [!tip]- 사용방법
>
> - **목록 이동**: 상단의 **목록 버튼 ①** 클릭 → 전체 회원 목록 화면으로 돌아간다.  
> - **정보 저장**: 변경 사항 입력 후 **저장 버튼 ②** 클릭 → 수정된 정보가 반영된다.  
> - **계정 복구**: **복구 버튼 ③** 클릭 → 휴면/탈퇴 회원을 정상 회원으로 전환한다.  
> - **계정 삭제**: **삭제 버튼 ④** 클릭 → 회원 계정을 영구 삭제한다.  
> - **질문 작성**: **질문하기 버튼 ⑤** 클릭 → 회원 관련 문의를 작성한다.  
> - **로그인 이력 확인**: **로그인 이력 버튼 ⑥** 클릭 → 회원의 접속 기록을 조회한다.  
> - **비밀번호 초기화**: **비밀번호 변경 버튼 ⑦** 클릭 → 새 비밀번호를 설정한다.  
> - **소속 지정**: **소속 선택 버튼 ⑧** 클릭 → 학습자/강사/조교 구분을 변경한다.  
> - **주소 등록**: 우편번호 입력 후 **주소찾기 버튼 ⑨** 클릭 → 주소를 자동 완성한다.
