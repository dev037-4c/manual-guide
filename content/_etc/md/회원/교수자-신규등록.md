> [!note]- 기능 설명
> #### 1. [목록](회원-교수자.md)
> 교수자 목록 화면으로 이동한다.
> 신규 등록 후 전체 현황을 다시 확인할 때 활용된다.
>
> #### 2. 저장  
> 입력한 교수자 정보를 저장한다.
> 등록 완료 후 시스템에 반영된다.
>
> #### 3. 변경내역  
> 교수자 정보의 변경 이력을 확인할 수 있다.
> 수정·관리 이력을 추적할 때 활용된다.
>
> #### 4. 중복검사  
> 입력한 ID가 기존에 존재하는지 확인한다.
> 중복 방지를 통해 고유 계정을 보장한다.
>
> #### 5. 소속 선택  
> 교수자의 소속을 지정한다.
> 부서·기관 단위로 관리할 때 사용된다.
> 
> #### 6. 주소찾기
> 우편번호 검색을 통해 주소를 자동으로 입력한다.
> 상세 주소는 별도로 직접 입력해야 한다.
> 
> #### 7. 프로필 사진 등록
> 교수자의 프로필 이미지를 업로드한다.
> 10MB 이하 파일만 등록 가능하다.
> 
> #### 8. 첨부파일 등록
> 이력서, 증명서 등 관련 자료를 첨부한다.
> 10MB 이하 파일만 업로드 가능하다.

---

> [!tip]- 사용 방법
> - **목록**: 상단의 **목록 버튼 ①** 클릭 → 교수자 목록 화면으로 돌아간다.
> - **저장**: 입력을 완료한 후 **저장 버튼 ②** 클릭 → 신규 교수자가 등록된다.
> - **변경내역**: 상단의 **변경내역 버튼 ③** 클릭 → 해당 교수자의 정보 변경 기록을 확인한다.
> - **중복검사**: ID 입력 후 **중복검사 버튼 ④** 클릭 → 동일 ID 존재 여부를 확인한다.
> - **소속 선택**: **소속 선택 버튼 ⑤** 클릭 → 드롭다운에서 해당 교수자의 소속을 선택한다.
> - **주소찾기**: **주소찾기 버튼 ⑥** 클릭 → 우편번호 입력 후 검색 → 자동으로 주소가 입력된다.
> - **프로필 사진 등록**: **파일선택 버튼 ⑦** 클릭 → 사진 파일 선택 후 업로드 → 화면에 이미지가 표시된다.
> - **첨부파일 등록**: **파일선택 버튼 ⑧** 클릭 → 이력서·증명서 등 문서 파일을 업로드한다.
