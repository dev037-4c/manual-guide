> [!note]- 기능설명
> 
> #### 1. [신규등록](menu관리-new-registration.md)
> 
> - 새로운 메뉴를 등록할 수 있다.
>     
> - 메뉴명, 페이지 유형, URL 등을 입력하여 생성한다.
>     
> 
> #### 2. [엑셀](%EC%97%91%EC%85%80.md)
> 
> - 메뉴 목록을 엑셀 파일로 내려받을 수 있다.
>     
> - 사이트 메뉴 구조를 문서화하거나 보고용으로 활용된다.
>     
> 
> #### 3. 삭제
> 
> - 선택한 메뉴를 삭제할 수 있다.
>     
> - 불필요하거나 중복된 메뉴를 정리할 때 사용된다.
>     
> 
> #### 4. [메뉴명](menu관리-페이지유형.md)
> 
> - 사이트의 메뉴명을 클릭하면 메뉴관리 화면능로 들어갈 수 있다.
>     
> - 상위/하위 메뉴 관계를 트리 형태로 확인할 수 있다.
>     
> 
> #### 5. [페이지 유형](menu관리-페이지유형.md)
> 
> - 각 메뉴가 연결된 URL 또는 직접 작성 여부를 확인할 수 있다.
>     
> - 클릭 시 해당 페이지로 이동하거나 설정 값을 확인할 수 있다.
>     
> 
> #### 6. [수정](menu관리-페이지유형.md)
> 
> - 등록된 메뉴를 수정할 수 있다.
>     
> - 아이콘 클릭 시 메뉴명, URL, 노출 여부 등을 변경할 수 있다.
>     

---

> [!tip]- 사용방법
> 
> - **신규등록**: 상단의 **신규등록 버튼 ①**을 클릭 → 새로운 메뉴를 등록한다.
>     
> - **엑셀 다운로드**: 상단의 **엑셀 버튼 ②**을 클릭 → 메뉴 목록을 저장한다.
>     
> - **삭제**: 불필요한 메뉴를 선택 후 **삭제 버튼 ③**을 클릭 → 목록에서 제거한다.
>     
> - **메뉴 구조 확인**: 중앙의 **메뉴 구조 ④**를 확인 → 트리 형태로 상위/하위 메뉴를 관리한다.
>     
> - **페이지 유형 확인**: 목록의 **페이지 유형 ⑤**을 클릭 → 연결된 URL 또는 직접 작성된 페이지를 확인한다.
>     
> - **수정**: 수정 아이콘 ⑥을 클릭 → 메뉴 정보를 변경한다.
>
