> [!note]- 기능설명
> #### 1. 탭
> - 상단 탭에서 언어(한국어, 영어, 중국어, 일본어)를 전환할 수 있다.
> - 선택한 언어에 따라 설정 화면이 해당 언어로 표시된다.
>
> #### 2. 저장
> - 서비스명, 로그인 페이지 문구, 로고, 푸터 등 변경된 항목을 저장한다.
> - 저장 시 최대 10분 이내 반영된다.
>
> #### 3. 파일선택
> - 로고 및 파비콘 이미지를 업로드할 수 있다.
> - 버튼 클릭 시 로컬 폴더 탐색기가 열리며 원하는 파일을 선택해 적용한다.

***

> [!tip]- 사용방법
> - **탭**: **탭 ①**클릭 시 원하는 언어 탭(한국어, 영어, 중국어, 일본어)을 클릭 → 해당 언어 화면으로 전환
> - **저장**: **저장 버튼 ②**을 클릭하면 서비스명, 로그인 페이지 문구, 로고, 푸터 등을 입력 또는 수정 → 저장 버튼 클릭 → 변경사항 반영
> -**파일 선택**: **파일 선택 버튼 ③**을 클릭하면 로고 또는 파비콘 업로드 영역에서 파일선택 버튼 클릭 → 로컬 PC에서 이미지 선택 → 적용 완료
