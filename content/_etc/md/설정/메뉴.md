***

> [!note]- 기능설명
> #### 1. 탭
> - 상단의 탭 메뉴에서 원하는 언어(한국어, 영어, 중국어, 일본어)를 클릭
> - 선택한 언어에 맞는 설정 화면으로 전환
>
> #### 2. [신규등록](학습-콘텐츠-신규등록.md)
> - 신규등록 버튼 클릭 시 메뉴 신규등록 페이지로 이동
> - 새로운 메뉴 추가할 때 사용
>
> #### 3. [엑셀](엑셀.md)
> - 엑셀 버튼 클릭 시 엑셀 다운로드 팝업 창 생성
> - 메뉴 목록을 엑셀 파일로 저장 가능
>
> #### 4. [메뉴명](메뉴명-상세.md)
> - 메뉴명 클릭 시 해당 메뉴명 상세 페이지로 이동
> - 메뉴명, URL, 노출여부 수정 가능
>
> #### 5. [페이지 유형](페이지유형-상세.md)
> - 페이지 유형 클릭 시 해당 페이지 유형 상세 페이지로 이동
> - 메뉴명, URL, 노출여부 수정 가능

***

> [!tip]- 사용방법
> - **탭 버튼 클릭** 시 상단 탭에서 언어를 클릭하면 해당 언어 설정 화면으로 전환됩니다.
> - **신규등록 버튼** 클릭 시 메뉴 신규등록 페이지로 이동하여 새 메뉴를 추가할 수 있습니다.
> - **엑셀 버튼 클릭** 시 메뉴 목록을 엑셀 파일로 다운로드할 수 있습니다.
> - **메뉴명을 클릭**하면 상세 페이지로 이동하여 메뉴명, URL, 노출 여부를 확인하고 수정할 수 있습니다.
> - **페이지 유형을 클릭**하면 상세 페이지로 이동하여 메뉴명, URL, 노출 여부를 확인하고 수정할 수 있습니다.
