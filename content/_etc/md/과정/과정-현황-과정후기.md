> [!note]- 기능 설명  
> 1. 과정유형 선택  
> - **상시제 / 기수제** 선택 가능  
> - **상시제**: 신청기간과 관계없이 학습자가 언제든 학습 가능한 과정  
> - **기수제**: 정해진 기간(기수 단위) 내 신청 및 학습 진행 과정  
>
> 2. 기간동일  
> - 신청기간, 수강취소기간, 학습기간을 **동일한 날짜로 자동 설정**  
> - 일괄 지정이 필요할 때 활용  
>
> 3. 개설과목 생성  
> - 입력 정보(과정유형, 기간 등)를 기반으로 새로운 **개설과목 생성**  
> - 생성된 과목은 [[과정 클릭 시]] 목록에서 확인 가능  

---

> [!tip]- 사용방법  
> - **과정유형 선택**: 상시제 / 기수제 선택  
> - **기간동일 설정**: 버튼 클릭 → 신청기간/수강취소기간/학습기간 자동 일치  
> - **개설과목 생성**: 개설과목 생성 버튼 클릭 → 새로운 개설과목 등록  
> - **생성 확인**: [과정 클릭 시] 목록 화면에서 생성된 개설과목 확인
