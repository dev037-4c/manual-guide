> [!note]- 기능 설명  
> #### 1. [엑셀](엑셀.md)  
> - 현재 화면에 표시된 학습 현황을 엑셀 파일로 다운로드한다.
> - 단순 조회, 내부 보고, 고객사 전달 자료로 활용된다.
>
> #### 2. 체크박스  
> - 학습 목록에서 여러 항목을 동시에 선택할 수 있다.
> - 삭제와 같은 일괄 작업을 수행할 때 사용된다.
>
> #### 3. 수료처리  
> - 선택한 학습자를 수료 상태로 변경한다.
> - 진도율과 시험/과제 조건 충족 시 활용된다.
>
> #### 4. 강제 수료처리
> - 학습 조건 충족 여부와 상관없이 운영자가 수료 처리할 수 있다.
> - 고객사 요청이나 특수 상황 시 사용된다.
>
> #### 5. 강제 미수료처리
> - 이미 수료된 학습자를 다시 미수료 상태로 전환한다.
> - 잘못된 수료 처리 수정이나 재학습 필요 시 활용된다.
>
> #### 6. 수강취소 
> - 선택된 학습자의 과정 참여를 취소 상태로 변경한다.
> - 학습자가 더 이상 해당 과정에 접근할 수 없게 된다.
>
> #### 7. 일자변경  
> - 학습자의 학습 시작일/종료일을 수정할 수 있다.
> - 운영자가 학습 기간을 유연하게 관리할 때 사용된다.
>
> #### 8. 검색
> - 검색창에서 특정 키워드(이름, ID 등)를 입력해 원하는 학습 목록을 찾을 수 있다.
>
> #### 9. [이름 / ID](개인정보.md)    
> - 학습자 ID 클릭 시 상세 페이지로 이동한다.
> - 학습자의 기본 정보 및 상세 내역 확인 가능하다.
>
> #### 10. [과정](과정관리-팝업창.md)
> - 과정명을 클릭하면 해당 과정의 관리 페이지로 이동한다.
> - 상세 페이지에서 학습자 관리, 성적 확인, 수료 처리 등을 진행할 수 있다.
> 
> #### 11. [진도율](과정-현황-학습-진도율클릭.md)
> - 학습자가 해당 과정을 얼마나 진행했는지 퍼센트(%)로 표시된다.
> - 진도율은 수료/미수료 판단의 핵심 지표로 활용된다.
> 
> #### 12. 시험
> - 학습자의 시험 응시 현황을 표시한다.
> - 응시 횟수/총 시험 수 형식으로 나타나며, 미응시 여부를 확인할 수 있다.
> 
> #### 13. 과제
> - 학습자의 과제 제출 현황을 표시한다.
> - 제출 건수/총 과제 수 형식으로 나타나며, 미제출 여부 확인이 가능하다.

---

> [!tip]- 사용방법  
> - **엑셀 다운로드**: 학습 현황을 저장하려면 **엑셀 버튼 ①** 클릭 → 저장 위치 선택 → 엑셀 파일 확인한다.
> - **학습자 선택**: 수료, 취소 등 작업을 하려면 **체크박스 ②** 클릭 → 원하는 학습자 선택 후 처리 버튼 클릭한다.
> - **수료 처리**: 조건 충족 학습자는 **수료처리 ③** 버튼, 조건 미충족이지만 강제 확정이 필요할 경우 **강제 수료처리 ④** 버튼을 사용한다.
> - **미수료 처리**: 잘못 처리된 건을 되돌리려면 **강제 미수료처리 ⑤** 클릭한다.
> - **수강 취소**: 학습자를 해당 과정에서 제외하려면 **수강취소 ⑥** 클릭한다.
> - **학습 기간 변경**: 운영자가 학습 일정을 수정하려면 **일자변경 ⑦** 클릭 → 시작일/종료일을 조정한다.
> - **검색 활용**: 특정 학습자를 찾으려면 **검색창 ⑧**에 이름/ID/과정명 입력 → 조건 조회 실행한다.
> - **이름/ID확인**: 학습자의 **이름/ID⑨**를 클릭 → 상세 페이지에서 기본 정보 및 학습 이력 확인한다.
> - **과정 확인**: 과정명을 보거나 수정하려면 **과정명 ⑩** 클릭 → 과정 상세 화면에서 관리 작업 수행한다.
> - **진도율 확인**: 학습자의 진행 상황은 **진도율 ⑪** 항목에서 퍼센트와 그래프를 통해 확인한다.
> - **시험 현황 확인**: 응시 여부를 확인하려면 **시험 ⑫** 항목 확인 → 응시 횟수/총 시험 수 비교한다.
> - **과제 현황 확인**: 제출 여부를 확인하려면 **과제 ⑬** 항목 확인 → 제출 건수/총 과제 수를 비교한다.
