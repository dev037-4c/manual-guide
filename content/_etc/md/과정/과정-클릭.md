***

> [!note]- 기능 설명
> #### 1. [목록](과정-패키지.md)
> - 패키지 목록 화면으로 이동한다.
> - 입력 중인 값은 저장되지 않으므로 이동 전 저장이 필요하다.
>
> #### 2. 저장
> - 입력한 패키지 정보를 등록하거나 수정 사항을 반영한다.
>
> #### 3. 삭제 
> - 선택된 패키지를 목록에서 삭제할 수 있다.
> - 체크박스로 선택한 항목만 삭제된다.
>
> #### 4. 과정 
> - 패키지 단위로 포함된 과정 관리 화면으로 이동한다.
>
> #### 5. 추가  
> - 새로운 과정을 패키지에 포함시킨다.
>
> #### 6. 엑셀
> - 현재 화면에 표시된 패키지 목록을 엑셀 파일로 내려받을 수 있다.
> - 단순 조회, 내부 보고, 고객사 전달 자료로 활용된다.
>
> #### 7. 체크박스
> - 패키지 목록에서 여러 항목을 동시에 선택할 수 있다.
> - 삭제와 같은 일괄 작업을 수행할 때 사용된다.
> 
> #### 8. 순서변경
> - 드래그 아이콘을 사용해 과정의 표시 순서를 변경할 수 있다.
>
> #### 9. ID
> - 과정 ID를 클릭하면 해당 과정의 상세 화면으로 이동한다.
> 
> #### 10. 제목
> - 과정명을 클릭하면 상세 화면으로 이동하여 정보를 확인하거나 수정할 수 있다.

---

>[!tip]- 사용방법
> - **목록**: 전체 현황을 보려면 **목록 버튼 ①** 클릭 → 패키지 목록 화면으로 이동한다.
> - **저장**: 신규 입력 또는 수정한 정보를 반영하려면 **저장 버튼 ②** 클릭 → 패키지 정보가 등록된다.
> - **삭제**: 불필요한 항목을 제거하려면 **삭제 버튼 ③** 클릭 → 확인 창에서 ‘확인’ 선택한다.
> - **과정 관리**: 포함된 과정을 관리하려면 **과정 버튼 ④** 클릭 → 과정 관리 화면으로 이동한다.
> - **추가**: 새 과정을 넣으려면 **추가 버튼 ⑤** 클릭 → 과정 선택 후 저장한다.
> - **엑셀 다운로드**: 포함된 과정 목록을 내려받으려면 **엑셀 버튼 ⑦** 클릭 → 저장 위치 선택 후 파일 생성된다.
> - **체크박스**: 개별 과정을 지정하려면 **체크박스 ⑦** 클릭 → 다중 선택 후 삭제 등 작업에 활용한다.
> - **순서 변경**: 과정 표시 순서를 바꾸려면 **순서변경 아이콘 ⑧** 드래그 → 원하는 위치로 이동한다.
> - **ID 확인**: 세부 정보를 확인하려면 **과정 ID ⑨** 클릭 → 상세 화면으로 이동한다.
> - **과정명 확인/수정**: 과정을 확인하거나 편집하려면 **제목 ⑩** 클릭 → 상세 화면에서 수정 후 저장한다.
