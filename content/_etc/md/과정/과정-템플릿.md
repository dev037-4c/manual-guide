> [!note]- 기능 설명  
> #### 1. [신규등록](과정-템플릿-신규등록.md)  
> - 새로운 템플릿을 생성할 수 있다.
> - 교육 과정 운영 시 반복적으로 활용할 수 있는 기본 틀을 등록하는 데 사용된다.
>
> #### 2. [엑셀](엑셀.md)  
> - 현재 화면의 템플릿 목록을 엑셀 파일로 내려받을 수 있다.
> - 단순 조회, 내부 보고, 고객사 전달 자료로 활용된다.
>
> #### 3. 체크박스  
> - 템플릿 목록에서 여러 항목을 동시에 선택할 수 있다.
> - 삭제와 같은 일괄 작업을 수행할 때 사용된다.
>
> #### 4. 삭제  
> - 선택된 템플릿을 목록에서 삭제할 수 있다.
> - 체크박스로 선택한 항목만 삭제된다.
>
> #### 5. 과정 수  
> - 각 템플릿에 연결된 과정의 개수를 보여준다.
> - 숫자를 클릭하면 연결된 과정의 목록이 열린다.
>
> #### 6. 제목
> - 제목을 클릭하면 수정 화면으로 이동하여 세부 내용을 변경할 수 있다.
>
> #### 7. 검색  
> - 검색창에서 특정 키워드(ID, 제목 등)를 입력해 원하는 템플릿을 찾을 수 있다.

---

> [!tip]- 사용방법  
> - **신규등록**: 새로운 템플릿을 만들려면 **신규등록 버튼 ①** 클릭 → 템플릿명, 설명, 필요한 항목 입력 → 저장한다.
> - **엑셀 다운로드**: 목록을 내려받으려면 **엑셀 버튼 ②** 클릭 → 저장 경로 선택 → 엑셀 파일 생성된다.
> - **템플릿 선택**: 다중 작업을 하려면 **체크박스 ③** 클릭 → 여러 템플릿을 동시에 지정한다.
> - **템플릿 삭제**: 필요 없는 항목은 **체크박스 ③**로 선택 → **삭제 버튼 ④** 클릭 → 확인 창에서 ‘확인’ 선택한다.
> - **연결 과정 확인**: 템플릿이 몇 개의 과정에 적용됐는지 보려면 **과정 수 ⑤** 클릭 → 연결된 과정 목록 확인한다.
> - **상세 확인**: 내용을 변경하려면 **제목 ⑥** 클릭 → 수정 화면에서 세부 정보 입력 후 저장한다.
> - **검색**: 원하는 템플릿을 찾으려면 **검색창 ⑦**에 ID나 제목 입력 → Enter 키로 조회한다.
