> [!note]- 기능설명
> #### 1. 목록
> - 클릭 시 콘텐츠 목록 페이지로 이동
>
> #### 2. 저장
> - 입력된 콘텐츠 정보를 저장
> - 필수 항목 미입력 시 저장 불가
>
> #### 3. 콘텐츠/파일 유형
> - 콘텐츠 유형(동영상, 오디오, PDF, HTML, 비메오, 유튜브, 웹에디터) 선택
> - 파일 유형(콘텐츠 원본, URL) 선택
>
> #### 4. 파일 선택
> - 파일 선택 버튼 클릭 시 파일 선택 창 생성
> - 포스터 이미지 등록도 가능
>
> #### 5. 자막
> - 자막 파일 등록 가능
> - AI 자막 자동 생성 가능
>
> #### 6. 재생시간 가져오기
> - 가져오기 버튼 클릭 시 콘텐츠 재생시간 확인 가능
>
> #### 7. 플레이어 설정
> - 이동바, 음소거, 음량 조절, 자막, 영상해속, 전체화면 등 기능 설정 가능
>
> #### 8. 운영권한
> - 전체, 등록자, 선택 항목으로 운영권한 설정 가능

***

> [!tip]- 사용방법
> - 콘텐츠를 등록하려면 상단의 **저장 버튼**을 클릭하기 전에 제목, 유형, 파일 등을 입력합니다.
> - 원하는 콘텐츠 유형(동영상, 오디오, PDF 등)과 파일 유형(원본, URL)을 **콘텐츠/파일 유형 항목**에서 선택합니다.
> - 콘텐츠 파일을 업로드하려면 **파일선택 버튼**을 클릭하여 파일을 등록합니다.
> - 자막이 필요한 경우 **자막 등록** 버튼을 통해 직접 파일을 업로드하거나, **AI 자막** 기능을 활용해 자동 생성할 수 있습니다.
> - 콘텐츠 재생시간을 확인하려면 **가져오기 버튼**을 클릭합니다.
> - 콘텐츠 표시 방식은 **플레이어 설정(이동바, 음소거, 자막 등)** 을 통해 조정할 수 있습니다.
> - 콘텐츠 운영 권한은 **전체, 등록자, 선택** 중 하나로 지정합니다.
> - 모든 항목 입력 후 **저장 버튼**을 클릭하면 등록이 완료됩니다.
> - 등록을 취소하거나 목록으로 돌아가려면 **목록 버튼**을 클릭합니다.
