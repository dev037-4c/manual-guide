> [!note]- 기능설명
> #### 1. 코드 다운로드
> - 현재 등록된 코드 목록을 엑셀 파일로 내려받을 수 있다.
> - 다운로드 시 즉시 엑셀 파일이 생성된다.
>
> #### 2. 템플릿 다운로드
> - 사전에 정의된 템플릿 양식을 엑셀 파일로 내려받을 수 있다.
> - 일괄 등록 시 사용할 표준 형식을 제공한다.
>
> #### 3. 파일선택
> - 로컬 PC에서 업로드할 엑셀 파일을 선택할 수 있다.
> - 버튼 클릭 시 파일 탐색기가 열리며, 원하는 파일을 지정한다.
>
> #### 4. 저장
> - 선택한 엑셀 파일을 시스템에 업로드하여 일괄 등록할 수 있다.
> - 등록 완료 시 코드 목록에 반영된다.

***

> [!tip]- 사용방법
> - **코드 다운로드**: 코드 목록을 내려받으려면 **다운로드 버튼 ①**을 클릭 → 엑셀 파일 저장.
> - **템플릿 다운로드**: 기본 양식을 내려받으려면 **템플릿 다운로드 버튼 ②**를 클릭한다.
> - **파일선택**: 업로드할 파일을 등록하려면 **파일선택 버튼 ③**을 클릭 → 로컬 폴더에서 엑셀 파일 선택.
> - **저장**: 선택한 파일을 일괄 등록하려면 **저장 버튼 ④**를 클릭한다.
