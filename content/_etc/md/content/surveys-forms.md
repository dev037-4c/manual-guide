> [!note]- 기능설명
> #### 1. [신규등록](surveys-forms-new-registration.md)
> - 새로운 설문지를 생성할 수 있다.
> - 클릭 시 설문지 등록 화면으로 이동하며, 문항 추가 및 옵션 설정이 가능하다.
>
> #### 2. [엑셀](excel.md)
> - 설문지 목록을 엑셀 파일로 다운로드할 수 있다.
> - 내부 보고서 작성이나 오프라인 분석 시 활용된다.
>
> #### 3. 검색
> - 설문지 목록에서 특정 키워드(ID, 제목 등)를 입력해 원하는 설문지를 찾을 수 있다.
>
> #### 4. [상세](surveys-forms-상세.md)
> - 설문지의 ID 또는 제목을 클릭하면 상세 페이지로 이동한다.
> - 상세 화면에서 설문지의 속성을 확인·수정할 수 있다.
>
> #### 5. [미리보기](surveys-forms-2.md)
> - 미리보기 버튼 클릭 시 팝업이 열리며 설문지가 실제 학습자 화면처럼 표시된다.
> - 설문지 등록 후 제공 화면을 사전에 검토할 수 있다.
>
> #### 6. [문항](surveys-forms-문항.md)
> - 설문지에 포함된 문항을 추가, 수정, 삭제할 수 있는 기능이다.
> - 문항 관리 페이지로 이동하여 편집할 수 있다.

***

> [!tip]- 사용방법
> - **신규등록**: 상단의 **신규등록 버튼 ①**을 클릭 → 설문 제목과 옵션을 입력 → 문항 추가 후 저장하여 설문지를 생성합니다.
> - **엑셀 다운로드**: **엑셀 버튼 ②**를 클릭 → 파일 저장 위치를 선택하면 현재 설문지 목록이 엑셀 파일로 내려받아집니다.
> - **검색**: 화면 우측 상단의 **검색창 ③**에 키워드를 입력 → 검색 버튼을 눌러 원하는 설문지를 조회합니다.
> - **상세 확인/수정**: 목록의 **ID 또는 제목 ④**을 클릭 → 상세 페이지에서 설문 제목, 구분, 사용 여부 등을 확인 및 수정합니다.
> - **미리보기**: 설문지를 학습자 화면에서 확인하려면 **미리보기 버튼 ⑤**를 클릭 → 팝업 창에서 설문 표시를 확인합니다.
> - **문항 관리**: 특정 설문지의 문항을 관리하려면 **문항 버튼 ⑥**을 클릭 → 문항 추가, 수정, 삭제 기능을 수행합니다.
