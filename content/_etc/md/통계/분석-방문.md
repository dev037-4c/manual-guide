> [!note]- 기능 설명  
> #### 1. 체크박스  
> - 방문 데이터를 확인할 분석 대상 페이지를 선택한다.
> - 예: 사용자메인페이지 체크 시 해당 페이지의 방문 수만 표시된다.
> 
> #### 2. 기간 선택  
> - 최근 단위 기간별 조회 기능을 제공한다.
> - 클릭 시 해당 기간의 월별 방문 데이터가 차트에 표시된다
> 
> #### 3. 기간 설정  
> - 시작월과 종료월을 직접 지정할 수 있다.
> - 조회 버튼 클릭 시 선택한 기간의 월별 방문 수가 표시된다.
> 
> #### 4. 상세 데이터  
> - 그래프 막대에 마우스를 올리면 해당 월의 방문 수 팝업이 표시된다.

---

> [!tip]- 사용방법  
> - **페이지 선택**: **체크박스 ①** 클릭 → 원하는 페이지 방문 데이터만 확인한다.
> - **기간 선택**: 상단의 **금년, 최근 1년, 최근 6개월, 최근 3개월 버튼 ②** 중 하나 클릭 → 차트에 해당 기간 데이터가 표시된다.
> - **기간 설정**: 시작월·종료월 입력 후 **조회 버튼 ③** 클릭 → 선택 기간의 방문 수 확인이 가능하다.
> - **상세 데이터**: 그래프 막대 위에 마우스 오버 → 해당 월의 방문 수 팝업 확인이 가능하다.
