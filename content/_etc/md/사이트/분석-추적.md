> [!note]- 기능설명
> #### 1번: 엑셀 버튼  
> 조회된 로그인 기록을 **엑셀 파일**로 다운로드할 수 있습니다.  
>
> #### 2번: 목록 테이블  
> 로그인 이력을 표 형태로 표시합니다. 주요 항목:  
> - **번호**: 로그 순번  
> - **로그인일시**: 로그인 발생 시간  
> - **사이트**: 로그인 구분 (사용자/관리자)  
> - **회원명 / ID**: 로그인한 계정 정보  
> - **로그인IP**: 접속한 IP 주소  
> - **세션ID**: 로그인 세션 식별 값  
>
> #### 3번: 검색 필터  
> 각 열(사이트, 회원명, ID, 로그인IP, 세션ID) 상단에 입력창이 있어 조건별로 검색 및 필터링 가능  
>
> #### 4번: 페이지 이동 및 항목수 설정  
> - **페이지 번호**: 하단에서 페이지 이동 가능  
> - **페이지 당 항목수**: 20/50/100개 등 표시 개수를 선택할 수 있음  
>
> #### 5번: 자동 새로고침 버튼  
> 최신 로그인 기록을 다시 불러오기 위해 화면 우측 하단의 새로고침 버튼 사용  

---

> [!tip]- 사용방법
> 1. 상단에서 필요한 조건(회원명, ID, 로그인IP 등)을 입력해 필터링합니다.  
> 2. 검색 조건을 지정하지 않으면 **금일 기준 로그인 기록**이 자동으로 표시됩니다.  
> 3. 특정 로그를 확인하거나 분석할 때는 **세션ID** 값을 확인합니다.  
> 4. 필요 시 **엑셀 버튼**을 눌러 다운로드하여 별도 보관/분석할 수 있습니다.  
> 5. 로그 수가 많을 경우 페이지 번호를 이동하거나 페이지당 항목수를 조정합니다.
