> [!note]- 기능설명
> #### 1. 목록
> - 과제 목록 화면으로 돌아가기 위한 버튼  
> - 저장하지 않고도 이동 가능  
>
>
> #### 2. 저장
> - 입력한 과제 정보를 저장  
> - 필수 항목이 입력되지 않으면 저장 불가  
>
> 
>
> #### 3. 삭제
> - 선택한 과제를 삭제  
> - 삭제 시 복구할 수 없으므로, 주의 필요  
>
>
> #### 4. [분류](분류.md)
> - 선택 버튼 클릭 시 분류선택 팝업 창 생성  
> - 하위단계 분류를 선택해야지만 등록 가능  
>
>
> #### 5. 파일선택
> - 로컬 폴더에서 업로드할 파일을 선택하는 기능  
> - 파일 선택 버튼 클릭 시 파일 탐색기가 열리며, 업로드할 엑셀 파일을 지정  
>
>
> #### 6. 내용
> - 글자 크기, 글꼴, 정렬 등 **서식 편집 기능**을 제공하여 자유롭게 문서 작성 가능  
> - 단락 구분, 목록(번호/기호), 표 삽입 등을 활용해 체계적인 내용 구성 가능  
> - 이미지나 첨부파일을 추가하여 과제 이해를 보조  
>

***

> [!tip]- 사용방법
> - 과제 목록 화면으로 돌아가려면 **목록 버튼**을 클릭합니다.  
> - 입력한 과제 정보를 저장하려면 **저장 버튼**을 클릭하며, 필수 항목이 입력되지 않으면 저장할 수 없습니다.  
> - 선택한 과제를 삭제하려면 **삭제 버튼**을 클릭하며, 삭제 시 복구가 불가능하므로 주의합니다.  
> - 분류를 등록하려면 **분류 선택 버튼**을 클릭하고 팝업에서 하위 분류를 선택합니다.  
> - 파일을 업로드하려면 **파일선택 버튼**을 클릭하고 로컬 폴더에서 엑셀 파일을 지정합니다.  
> - 과제 내용을 작성하거나 편집하려면 **내용 편집 기능**을 활용하여 글꼴, 정렬, 표 삽입, 이미지 첨부 등을 적용합니다.
