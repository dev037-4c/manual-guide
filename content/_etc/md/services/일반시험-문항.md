> [!note]- 기능 설명  
> #### 1. 신규등록  
> - 새로운 시험 문항을 추가할 수 있다.
> - 문제 유형(객관식, 주관식 등), 보기, 정답, 난이도 등을 직접 입력한다.
>  
> #### 2. [엑셀](excel.md)
> - 현재 조회된 문항 목록을 엑셀 파일로 내려받는다.
> - 단순 조회, 내부 보고, 고객사 전달 자료로 활용된다.
>  
> #### 3. 일괄등록  
> - 다수의 문항을 엑셀 양식을 통해 한 번에 등록할 수 있다.
> - 대량 문항 관리 시 효율적으로 사용할 수 있다.
>  
> #### 4. [ID / 제목](services-일반exams-questions-ID_제목클릭.md)
> - 문항 ID / 제목을 클릭하면 상세 페이지로 이동한다.
> - 세부 내용, 보기, 난이도를 확인하거나 수정할 수 있다.
>  
> #### 5. [미리보기](services-일반exams-questions-미리보기.md)  
> - 해당 문항을 실제 응시자 화면과 동일하게 확인할 수 있다.
> - 보기 표시, 정답 노출 방식 등을 사전에 검토할 수 있다.
>  
> #### 6. 검색  
> - 검색창에 특정 키워드(ID, 제목 등)를 입력해 원하는 문항을 찾을 수 있다.

---

> [!tip]- 사용방법  
> - **신규등록**: **신규등록 버튼 ①** 클릭 → 문제 유형, 내용, 정답 입력 → 저장하여 등록한다.
> - **엑셀 다운로드**: **엑셀 버튼 ②** 클릭 → 저장 위치 선택 → 현재 목록을 엑셀 파일로 내려받는다.
> - **일괄등록**: **일괄등록 버튼 ③** 클릭 → 지정된 엑셀 템플릿에 문항 작성 → 업로드하여 다수 문항을 한 번에 등록한다.
> - **ID/제목**: 목록의 **제목 ④** 클릭 → 상세 페이지에서 보기, 난이도 수정 → 저장 시 반영된다.
> - **미리보기**: 각 행의 **미리보기 버튼 ⑤** 클릭 → 응시자 화면 그대로의 문항 표시를 확인한다.
> - **검색**: 상단의 **검색창 ⑥**에 ID 또는 제목 입력 → 조회 버튼 클릭 → 조건에 맞는 문항만 표시된다.
