***

> [!note]- 기능설명
> #### 1. 목록
> - 시험지 목록 화면으로 돌아가기 위한 버튼
> - 저장하지 않고도 이동 가능
>
> #### 2. 저장
> - 시험지의 내용을 수정한 후 반드시 클릭 시 변경 사항 저장
> - 문항 수정, 설명 추가, 사용 여부 변경 등 모두 저장 대상
>
> #### 3. [분류](분류.md)
> - 선택 버튼 클릭 시 분류선택 팝업 창 생성
> - 하위단계 분류를 선택해야지만 등록 가능
>
> #### 4. [과정 템플릿](과정템플릿.md)
> - 선택 버튼 클릭 시 과정템플릿 선택 팝업 창 생성
>
> #### 5. [문항](시험-시험지-신규등록_문항.md)
> - 과정 템플릿을 선택해야지만 추가 가능
> - 템플릿에 있는 목록 선택 시 문항 내용 추가
>
> #### 6. 운영권한 설정
> - 시험지에 대한 접근 및 운영 권한을 설정

***

> [!tip]- 사용방법
> - **목록 버튼** 클릭 시 시험지 목록 화면으로 이동합니다. (저장하지 않아도 이동 가능)
> - **저장 버튼** 클릭 시 시험지의 수정·추가·변경 사항 저장합니다.
> - **분류 선택 버튼** 클릭 시 팝업 창에서 하위 단계 분류 선택 후 등록 가능합니다.
> - **과정 템플릿 선택 버튼** 클릭 시 팝업 창에서 과정 템플릿 선택합니다.
> - **문항 추가**는 과정 템플릿을 선택해야 가능하며, 선택 시 문항 자동 추가합니다.
> - **운영권한 설정**에서 시험지 접근 및 운영 권한 지정합니다.
