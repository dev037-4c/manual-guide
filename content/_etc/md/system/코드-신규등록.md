> [!note]- 기능설명  
> #### 1. 목록
> - 코드 목록 화면으로 이동한다.  
> - 작성 중인 신규 코드 입력은 저장하지 않아도 이동할 수 있다.  
>
> #### 2. 저장  
> - 입력한 코드 정보를 신규로 등록한다.  
> - 그룹코드명, 그룹코드 등 필수 항목을 저장해야 등록된다.  
>
> #### 3. 다국어 코드  
> - 한국어, 영어, 중국어, 일본어 등 언어별 코드명을 등록할 수 있다.  
> - 다국어 환경에서 동일한 코드를 여러 언어로 지원한다.  

---

> [!tip]- 사용방법  
> - **목록**: 신규 코드 입력 중 목록으로 돌아가려면 **목록 버튼 ①** 클릭.  
> - **저장**: 필수 항목 입력 후 **저장 버튼 ②** 클릭 → 코드가 신규 등록된다.  
> - **다국어 코드 입력**: 필요 시 **다국어 코드 버튼 ③** 클릭 → 언어별 코드명을 입력·저장한다.
