> [!note]- 기능설명
> #### 1. 다운로드
> - 코드 목록을 엑셀 파일로 내려받는 기능  
> - 다운로드 버튼 클릭 시 현재 등록된 코드 목록이 엑셀 파일로 생성  
>
> ***
>
> #### 2. 파일선택
> - 로컬 폴더에서 업로드할 파일을 선택하는 기능  
> - 파일 선택 버튼 클릭 시 파일 탐색기가 열리며, 업로드할 엑셀 파일을 지정  
>
> ***
>
> #### 3. 템플릿 다운로드
> - 버튼 클릭 시 사전에 정의된 템플릿 엑셀 파일 다운로드  
>
> ***
>
> #### 4. 저장
> - 선택한 파일을 시스템에 일괄 등록하는 기능  
> - 저장 버튼 클릭 시 선택된 엑셀 파일 일괄 등록  
>
> ***

***

> [!tip]- 사용방법
> - 코드 목록을 내려받으려면 **다운로드 버튼**을 클릭하여 엑셀 파일로 저장합니다.  
> - 업로드할 파일을 등록하려면 **파일선택 버튼**을 클릭하고 로컬 폴더에서 엑셀 파일을 지정합니다.  
> - 템플릿을 내려받으려면 **템플릿 다운로드 버튼**을 클릭하여 기본 양식을 엑셀 파일로 저장합니다.  
> - 선택한 파일을 일괄 등록하려면 **저장 버튼**을 클릭해 시스템에 업로드합니다.
