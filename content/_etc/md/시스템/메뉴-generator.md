> [!note]- 기능설명  
> #### 1. 검색  
> - 데이터베이스 사용자 ID를 입력해 연결된 테이블 목록을 조회할 수 있다.
> - 검색 버튼을 클릭하면 해당 UserID에 속한 모든 테이블이 화면에 표시된다.  
> 
> #### 2. 프로젝트 설정
> - Project에는 시스템 구분명을, Package에는 소스 경로를, Component에는 기능 단위를 입력한다.
> - 생성자 입력란에는 담당자 이름을 입력하며, 생성할 파일 유형을 선택할 수 있다.
> 
> #### 3. 생성
> - 선택한 테이블과 입력된 설정 정보를 기반으로 자동으로 코드 파일을 생성한다.
> - 생성 버튼을 클릭하면 지정한 경로와 패키지 구조에 맞춰 코드가 자동 생성된다.

---

> [!tip]- 사용방법  
> - **Database UserID 입력**: 상단의 **입력란①**에 DB 사용자 ID를 입력 →검색 버튼을 클릭 →해당 계정의 테이블 목록이 표시된다.
> - **프로젝트 설정**: **하단 영역②**에서 생성자 정보를 입력 →생성할 파일 유형을 체크한다.
> - **코드 생성**: 모든 설정을 완료 후 → **생성 버튼③**을 클릭 → 선택한 테이블 기준으로 코드 파일이 자동으로 생성된다.
