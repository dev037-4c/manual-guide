> [!note]- 기능설명  
> #### 1. [목록](이벤트-당첨자발표.md)  
> - 작성된 당첨자 발표 전체 목록으로 이동한다.  
>
> #### 2. 저장  
> - 입력한 제목, 대상, 기간, 내용 등을 저장한다.  
>
> #### 3. 삭제  
> - 현재 작성 중인 당첨자 발표 글을 삭제한다.  
>
> #### 4. 미리보기  
> - 작성한 내용을 실제 게시 화면처럼 미리 확인할 수 있다.  
>
> #### 5. URL 복사  
> - 해당 당첨자 발표 글의 고유 URL을 클립보드에 복사한다.  
>
> #### 6. 파일선택  
> - 당첨자 발표와 함께 첨부할 파일을 업로드할 수 있다.  

---

> [!tip]- 사용방법  
> 1. **목록**: **목록 버튼 ①** 클릭 → 당첨자 발표 전체 목록 화면으로 이동한다.  
> 2. **저장**: **저장 버튼 ②** 클릭 → 입력한 제목, 대상, 기간, 내용 등이 저장된다.  
> 3. **삭제**: **삭제 버튼 ③** 클릭 → 현재 작성 중인 당첨자 발표 글이 삭제된다.  
> 4. **미리보기**: **미리보기 버튼 ④** 클릭 → 실제 게시되는 화면처럼 내용을 확인한다.  
> 5. **URL 복사**: **URL 복사 버튼 ⑤** 클릭 → 해당 글의 고유 주소가 클립보드에 저장된다.  
> 6. **파일선택**: **파일선택 버튼 ⑥** 클릭 → 첨부파일 업로드 창에서 원하는 파일을 추가한다.
