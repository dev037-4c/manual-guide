> [!note]- 기능 설명  
> 1. 신규등록  
> - 새로운 시험지를 등록할 때 사용  
> - `[신규등록]` 버튼 클릭 시 시험지 신규등록 페이지로 이동  
> - 시험명, 출제 방식(랜덤/지정), 문항 추가 기능 등을 입력하여 등록  
>  
> 2. 엑셀  
> - 현재 조회된 시험지 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
>  
> 3. 제목 클릭  
> - 시험지 제목(ID, 명칭)을 클릭하면 **상세 페이지**로 이동  
> - 상세 화면에서 시험명, 출제 유형, 문항, 사용 여부, 단계별 분류 등을 확인 및 수정  
>  
> 4. 미리보기  
> - 시험지 화면을 실제 응시자 화면과 동일하게 **미리보기** 가능  
> - 문항, 난이도, 랜덤 출제 여부 등을 점검  
> - 클릭 시 팝업 또는 새 창에서 미리보기 제공  
>  
> 5. 검색  
> - 원하는 시험지를 빠르게 찾기 가능  
> - 검색 기준: 시험지 ID, 제목, 단계 등  
> - 검색어 입력 후 조회 시 해당 시험지만 표시  

---

> [!tip]- 사용방법  
> - **신규 시험지 등록**: `[신규등록]` 버튼 클릭 → 시험 정보 입력 → 저장  
> - **시험지 목록 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **시험지 상세 확인/수정**: 제목 클릭 → 상세 페이지에서 수정  
> - **시험지 미리보기**: `[미리보기]` 버튼 클릭 → 응시자 화면과 동일하게 확인  
> - **특정 시험 검색**: 검색창에 ID, 제목, 단계 등 입력 → `[조회]` 클릭
