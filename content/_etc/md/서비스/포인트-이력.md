> [!note]- 기능 설명  
> 1. 엑셀  
> - 현재 조회된 포인트 이력을 엑셀 파일로 다운로드할 수 있음  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
> 2. 검색  
> - **검색창**에 키워드를 입력하여 포인트 이력을 조회할 수 있음  
> - ID, 이름, 내용 등 조건으로 검색이 가능  
> 3. 구분  
> - 포인트 이력을 **적립 / 차감** 기준으로 필터링  
> - 원하는 구분을 선택하면 해당 조건에 맞는 데이터만 표시  
> 4. 일자  
> - 특정 기간에 해당하는 포인트 이력을 조회  
> - 시작일과 종료일을 선택하여 원하는 기간의 데이터만 확인 가능  

---

> [!tip]- 사용방법  
> - **포인트 이력 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록 저장  
> - **특정 회원 조회**: 검색창에 ID 또는 이름 입력 → `[조회]` 클릭  
> - **적립/차감 기준 조회**: 구분 필터에서 조건 선택 → `[조회]` 클릭  
> - **기간별 조회**: 일자 필터에서 시작일과 종료일 설정 → `[조회]` 클릭
