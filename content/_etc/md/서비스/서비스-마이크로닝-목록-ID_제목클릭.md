> [!note]- 기능설명  
> #### 1. [목록](마이크로러닝-목록.md)    
> - 등록/수정 화면에서 전체 과정 목록으로 돌아간다.  
> - 다른 과정을 다시 확인하거나 관리할 수 있다.  
>
> #### 2. 저장  
> - 입력한 과정 정보를 저장한다.  
> - 신규 등록 또는 수정한 내용이 반영된다.  
>
> #### 3. 삭제  
> - 불필요하거나 잘못 입력된 과정을 삭제한다.  
> - 삭제 후에는 복구가 불가능하므로 신중히 사용한다.  
>
> #### 4. 수정  
> - 기존 등록된 과정 정보를 편집할 수 있다.  
> - 과정명, 설명, 학습 대상 등을 변경할 때 사용한다.  
>
> #### 5. 목차 설정  
> - 과정 내 학습 차시와 주차를 구성하는 화면으로 이동한다.  
> - 학습 진행 순서를 관리할 수 있다.  
>
> #### 6. 분류 선택  
> - 과정이 속할 분류를 선택한다.  
> - 지정된 분류를 통해 체계적으로 관리할 수 있다.  

---

> [!tip]- 사용방법  
> - **목록**: 상단의 **목록 버튼①** 클릭 → 전체 과정 목록 화면으로 이동한다.  
> - **저장**: 입력한 내용을 반영하려면 **저장 버튼②** 클릭 → 신규 등록 또는 수정 사항이 저장된다.  
> - **삭제**: 불필요한 항목을 제거하려면 **삭제 버튼③** 클릭 → 확인 창에서 ‘확인’을 눌러 삭제한다.  
> - **수정**: 기존 과정을 편집하려면 **수정 버튼④** 클릭 → 변경할 항목 입력 후 저장한다.  
> - **목차 설정**: 학습 구조를 관리하려면 **목차 설정 버튼⑤** 클릭 → 차시·주차를 구성한다.  
> - **분류 선택**: 과정의 분류를 지정하려면 **선택 버튼⑥** 클릭 → 원하는 분류를 선택 후 반영한다.
