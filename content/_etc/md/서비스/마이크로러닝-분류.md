> [!note]- 기능 설명  
> #### 1. [신규등록](서비스-마이크로닝-분류-신규등록.md)  
> - 새로운 마이크로러닝 분류를 생성한다.
> - 상위/하위 구조로 트리를 구성해 체계적으로 관리한다.
>  
> #### 2. [엑셀](엑셀.md)  
> - 현재 화면의 분류 목록을 엑셀 파일로 다운로드한다. 
> - 단순 조회, 내부 보고, 고객사 전달 자료로 활용된다.
> 
> #### 3. 삭제
> - 선택된 분류 목록에서 삭제할 수 있다.
> - 체크박스로 선택한 항목만 삭제된다.
>
> #### 4. [분류명](서비스-마이크로닝-분류-분류명_수정클릭.md) 
> - 폴더 아이콘(▾/▸)으로 접기/펼치기 하여 상·하위 구조를 확인한다.
>
> #### 5. 체크박스
> - 분류 목록에서 여러 항목을 동시에 선택할 수 있다.
> - 삭제와 같은 일괄 작업을 수행할 때 사용된다.
> 
> #### 6. [수정](서비스-마이크로닝-분류-분류명_수정클릭.md)
> - 톱니바퀴 아이콘으로 상세/수정 화면으로 이동한다.
>
> #### 7. [순서변경](순서클릭.md)
> - 표시 정렬 순서를 설정한다.

---

> [!tip]- 사용방법
> - **신규등록**: **신규등록 버튼 ①** 클릭 → 분류명, 상위 분류, 코드 입력 → **저장**으로 생성한다.
> - **엑셀 다운로드**: **엑셀 버튼 ②** 클릭 → 저장 위치 선택 → 현재 목록을 엑셀로 내려받는다.
> - **검색**: 상단 **분류명/코드 입력란 ③**에 키워드 입력 → Enter 또는 검색 버튼으로 결과를 필터링한다.
> - **삭제**: 삭제할 항목을 **체크박스 ⑥**로 선택 → **삭제 버튼 ④** 클릭 → 확인 창에서 **확인**을 눌러 삭제한다.
> - **분류 트리 확인**: **트리 토글 ⑤**(▾/▸) 클릭 → 하위 분류를 펼치거나 접어 구조를 확인한다.
> - **수정**: 대상 행의 **톱니바퀴 ⑦** 클릭 → 상세 화면에서 분류명, 코드, 상위 분류 수정 후 **저장**한다.
> - **순서 변경**: 대상 행의 **순서 ⑧**에서 값 입력 또는 화살표로 조정 → 변경 사항이 즉시 반영되지 않으면 화면의 **저장/적용** 버튼을 눌러 완료한다.
