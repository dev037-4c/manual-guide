> [!note]- 기능설명  
> #### 1. [목록](마이크로러닝-목록.md)  
> - 신규등록 화면에서 목록으로 이동한다.  
> - 전체 목록을 다시 확인하거나 다른 목록을 선택할 수 있다.  
>
> #### 2. 저장  
> - 신규 등록 또는 수정된 내용이 반영된다.  
>
> #### 3. 삭제  
> - 선택한 목록을 삭제한다.  
> - 확인 절차 후 복구는 불가능하다.  
>
> #### 4. 수정  
> - 기존에 등록된 내용을 편집할 수 있다.  
> - 제목, 분류, 학습대상, 이미지 등을 변경한다.  
>
> #### 5. 목차 설정  
> - 과정 내 학습 차시·주차 구성을 설정한다.  
> - 학습자의 학습 흐름을 관리할 수 있다.  
>
> #### 6. 검색  
> - 과정명 입력 후 **검색 버튼**을 눌러 원하는 과정을 조회한다.  
>
> #### 7. 썸네일 업로드  
> - 대표 이미지를 등록할 수 있다. (권장 크기 480×270)  
>
> #### 8. 첨부자료 업로드  
> - 과정 운영에 필요한 참고 문서를 업로드한다.  

---

> [!tip]- 사용방법  
> - **신규등록**: 상단의 **신규등록 버튼①** 클릭 → 과정명, 분류, 학습대상, 키워드 입력 → 저장 버튼으로 등록한다.  
> - **저장**: 모든 입력값을 확인 후 **저장 버튼②** 클릭 → 신규 목록이 등록된다.  
> - **삭제**: 불필요한 경우 **삭제 버튼③** 클릭 → 확인 창에서 ‘확인’을 눌러 삭제한다.  
> - **수정**: 기존 과정을 편집하려면 목록에서 항목 선택 후 **수정 버튼④** 클릭 → 원하는 내용을 변경 후 저장한다.  
> - **목차 설정**: 과정의 세부 학습 구조를 관리하려면 **목차 설정 버튼⑤** 클릭 → 주차, 차시 등을 구성한다.  
> - **검색**: 특정 과정을 찾으려면 과정명 입력 후 **검색 버튼⑥** 클릭 → 조건에 맞는 항목이 조회된다.  
> - **썸네일 업로드**: 과정 대표 이미지를 등록하려면 **파일선택 버튼⑦** 클릭 → 이미지 파일을 업로드한다.  
> - **첨부자료 업로드**: 참고 자료를 등록하려면 **파일선택 버튼⑧** 클릭 → 파일을 첨부한다.
