> [!note]- 기능 설명  
> 1. 신규등록  
> - 새로운 설문을 생성할 때 사용  
> - `[신규등록]` 버튼 클릭 시 **설문 신규등록 페이지**로 이동  
> - 설문 제목, 설문 기간, 문항 추가 등 등록  
>  
> 2. 체크박스  
> - 설문 목록에서 **개별 설문 선택** 시 사용  
> - 다중 선택 가능  
> - 선택 시 상단의 **삭제 버튼(3번)**이 활성화  
>  
> 3. 삭제  
> - 체크박스로 선택한 설문을 삭제  
> - 삭제 시 해당 설문은 목록에서 제거  
> - ⚠️ **주의: 이미 진행된 설문의 데이터는 복구 불가능**  
>  
> 4. 제목  
> - 설문 **제목(ID)** 클릭 시 상세 페이지로 이동  
> - 설문 제목, 기간, 상태, 문항 등을 확인 및 수정  
>  
> 5. 미리보기  
> - 설문 응답자 화면을 사전에 확인할 수 있는 기능  
> - 클릭 시 **실제 설문지 화면과 동일한 뷰**를 제공  
> - 설문 배포 전 문항 확인 및 UI 검수가 가능  
>  
> 6. 검색  
> - 특정 설문을 빠르게 찾을 수 있는 기능  
> - 검색 기준: 설문 ID, 제목 등  
> - 원하는 키워드를 입력 후 조회 시 해당 설문만 목록에 표시  

---

> [!tip]- 사용방법  
> - **새 설문 생성**: `[신규등록]` 버튼 클릭 → 설문 기본 정보 입력 → 저장  
> - **설문 삭제**: 체크박스 선택 후 `[삭제]` 버튼 클릭  
> - **설문 상세 수정**: 설문 ID 또는 제목 클릭 → 상세 페이지 이동  
> - **설문 미리보기**: `[미리보기]` 버튼 클릭 → 실제 설문지 확인  
> - **설문 검색**: 검색창에 키워드 입력 후 `[조회]` 클릭
