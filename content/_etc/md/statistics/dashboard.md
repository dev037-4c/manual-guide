> [!note]- 기능 설명  
> #### 1. 기간 선택 
> - 조회 기간을 빠르게 설정할 수 있다.
> - 클릭 시 해당 기간의 수료자 수 / 과정 등록 현황이 자동으로 차트에 반영된다.
> 
> #### 2. 기간 직접 설정  
> - 시작일자와 종료일자를 직접 지정할 수 있다.
> - 설정한 기간의 데이터가 차트와 표에 표시된다.
> 
> #### 3. 상세 데이터  
> - 차트의 특정 막대 위에 마우스를 올리면 해당 시점의 세부 수치가 툴팁으로 표시된다.
> - 수료자 수, 과정 유형별 등록 현황 등 데이터를 직관적으로 확인할 수 있다.

---

> [!tip]- 사용방법  
> - **기간 선택**: 상단의 **기간 선택 버튼 ①**을 클릭하면 설정된 기간의 통계 차트가 표시된다.
> - **기간 직접 설정**: 시작일과 종료일을 직접 입력한 후 **조회 버튼 ②**을 클릭하면 선택한 기간의 데이터가 반영된다.
> - **상세 데이터**: 차트 막대 위에 마우스를 올리면 해당 월의 수료자 수와 과정 등록 현황이 표시된다.
