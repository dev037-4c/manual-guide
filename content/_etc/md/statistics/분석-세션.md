> [!note]- 기능 설명  
> #### 1. 체크박스  
> - 분석할 대상 페이지를 선택할 수 있다.
> - 복수 선택이 가능하여 여러 페이지 데이터를 비교할 수 있다.
> 
> #### 2. 기간 선택  
> - 금년, 최근 1년, 최근 6개월, 최근 3개월 등 단위 기간별 조회 기능을 제공한다.
> - 클릭 시 선택한 기간의 월별 세션 데이터가 차트에 반영된다.
> 
> #### 3. 기간 설정  
> - 시작일과 종료일을 직접 지정할 수 있다.
> - 조회 버튼을 클릭하면 선택한 기간의 월별 세션 데이터가 반영된다.
> 
> #### 4. 상세 데이터  
> - 그래프 막대에 마우스를 올리면 해당 월의 세션 수가 팝업으로 나타난다.

---

> [!tip]- 사용방법  
> - **체크박스**: 분석할 페이지 유형 **체크박스 ①** 선택 → 해당 페이지의 세션만 확인한다.
> - **기간 선택**: 상단의 **금년/최근 1년/최근 6개월/최근 3개월 버튼 ②** 클릭 → 차트에 해당 기간 세션 데이터가 반영된다.
> - **기간 설정**: 시작일·종료일 입력 후 **조회 버튼 ③** 클릭 → 지정한 기간의 세션 데이터를 확인한다.
> - **상세 데이터**: 그래프 막대 위에 마우스를 올리면 **팝업 ④**에서 해당 월의 세션 수 확인이 가능하다.
