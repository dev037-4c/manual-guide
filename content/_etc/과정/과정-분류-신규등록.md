> [!note]- 기능 설명  
> 1. [목록](과정-분류.md)  
> - 분류 **목록 화면**으로 이동  
> - 현재 입력 내용은 저장되지 않으므로 이동 전 저장 권장  
>
> 2. 저장  
> - 입력한 **상위분류, 분류코드, 분류명** 정보 저장  
> - 저장 후 목록에서 새로 등록된 분류 확인 가능  
>
> 3. 선택  
> - 상위분류 선택 팝업 창 열림  
> - 다단계 구조(1~4단계)에서 원하는 상위 카테고리 지정 가능  

---

> [!tip]- 사용방법  
> - **분류명/분류코드 입력**: 신규 분류 정보 입력  
> - **상위 분류 지정**: `[선택]` 버튼 클릭 → 팝업 창에서 상위 카테고리 선택  
> - **저장**: `[저장]` 버튼 클릭 → 신규 분류 등록 완료  
> - **등록 확인**: `[목록]` 화면 이동 → 생성된 분류 정상 등록 여부 확인  
> - **추가 등록**: 동일 절차 반복 → 다른 분류 연속 등록 가능
