> [!note]- 기능 설명  
> 1. [신규등록](과정-분류-신규등록.md)  
> - 새로운 과정 **분류 추가**  
> - 클릭 시 **분류 등록 화면**으로 이동  
>
> 2. [엑셀](엑셀.md)  
> - 화면의 분류 목록을 **엑셀 파일로 다운로드**  
> - 단순조회, 내부 보고, 고객사 전달용 활용  
>
> 3. 삭제  
> - 체크박스로 선택한 분류 삭제  
> - 삭제 전 확인 팝업 표시 → **삭제 후 복구 불가**  
>
> 4. 분류 선택 체크박스  
> - 특정 분류 항목 선택 가능  
> - 선택 후 **[삭제]** 기능과 다중 선택 작업 활용  
>
> 5. 제목 클릭  
> - 분류 제목 표시  
> - 제목 클릭 시 **수정 화면**으로 이동  

---

> [!tip]- 사용방법  
> - **신규 등록**: `[신규등록]` 버튼 클릭 → 분류 정보 입력 → 저장  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 현재 분류 목록 저장  
> - **분류 삭제**: 체크박스 선택 → `[삭제]` 버튼 클릭 → 확인 후 삭제  
> - **다중 선택**: 체크박스 여러 개 선택 → 일괄 삭제 등 작업 가능
