> [!note]- 기능설명  
> #### 1. [신규등록](과정-분류-신규등록.md)  
> 새로운 과정을 분류에 추가하는 기능이다.  
> 클릭 시 분류 등록 화면으로 이동하며, 교육 과정의 관리 체계를 확장할 때 활용된다.  
>
> #### 2. [엑셀](엑셀.md)  
> 현재 화면의 분류 목록을 엑셀 파일로 내려받는 기능이다.  
> 단순 조회, 내부 보고, 고객사 전달 자료 생성 시 유용하다.  
>
> #### 3. 삭제  
> 체크박스로 선택한 분류를 삭제하는 기능이다.  
> 삭제 시 확인 창이 표시되며, 완료 후에는 복구가 불가능하다.  
>
> #### 4. 분류 선택 체크박스  
> 특정 분류 항목을 선택하는 기능이다.  
> 다중 선택이 가능하며, 삭제 등 일괄 작업에 활용된다.  
>
> #### 5. 제목 클릭  
> 분류 제목을 클릭하면 수정 화면으로 이동한다.  
> 기존 분류 정보를 확인·수정할 때 사용된다.  

---

> [!tip]- 사용방법  
> - **신규등록**: `[신규등록]` 버튼 클릭 → 분류명, 설명 입력 → 저장  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 저장 위치 선택 → 목록 파일 생성  
> - **분류 삭제**: 체크박스 선택 → `[삭제]` 버튼 클릭 → 확인 창에서 ‘확인’ 선택  
> - **다중 선택**: 여러 항목을 체크박스로 선택 → 일괄 삭제 등 작업 수행
