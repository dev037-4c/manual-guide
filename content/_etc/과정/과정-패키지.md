## 기능 설명  

> [!note]- #### 1. [신규등록](패키지등록.md)  
> - 새로운 **패키지** 등록.  
> - 클릭 시 **패키지 등록 화면**으로 이동.  

> [!note]- #### 2. [엑셀](엑셀.md)  
> - 현재 화면의 패키지 목록 **엑셀 다운로드**.  
> - 단순조회, 내부 보고, 고객사 전달용 활용.  

> [!note]- #### 3. 체크박스  
> - 개별 패키지 선택 기능.  
> - 다중 선택 시 **일괄 작업, 삭제** 등 상단 메뉴 활용 가능.  

> [!note]- #### 4. 삭제  
> - 선택한 패키지 삭제.  
> - 삭제 전 확인 메시지 표시, 삭제 후 복구 불가.  

> [!note]- #### 5. 패키지 제목  
> - 패키지명 표시.  
> - 클릭 시 **상세/수정 화면**으로 이동, 기본 정보 및 연결 과정 편집 가능.  

> [!note]- #### 6. 과정 수  
> - 패키지에 포함된 과정 수 표시.  
> - 숫자 클릭 시 연결된 과정 목록 화면으로 이동.  

> [!note]- #### 7. 검색  
> - 패키지명, 등록자, 기간 등 조건 검색.  
> - 조건 검색을 통해 빠른 조회 가능.  

---

> [!tip]  
> ## 사용방법  
> - **신규 등록**: `[신규등록]` 클릭 → 패키지 정보 입력 → 저장.  
> - **일괄 작업**: 체크박스 선택 → `[일괄]` 클릭 → 일괄 승인/삭제 처리.  
> - **개별 선택**: 체크박스로 패키지 선택 → 상단 메뉴 실행(삭제, 일괄 등).  
> - **패키지 삭제**: 체크박스 선택 → `[삭제]` 클릭 → 확인 메시지 후 삭제 완료.  
> - **상세 확인**: 패키지명 클릭 → 패키지 상세/수정 화면 이동.  
> - **과정 목록 확인**: 과정 수 숫자 클릭 → 연결된 과정 목록 조회.  
> - **검색 활용**: 검색창에 조건 입력(패키지명, 등록자, 기간 등) → `[조회]` 클릭.
