## 기능 설명  

> [!note]- 기능 설명  
> #### 1. [신규등록](기수등록.md)  
> - 새로운 **기수** 생성함  
> - 클릭 시 기수 등록 화면으로 이동 → **기수명, 적용기간, 신청기간, 과정** 설정 가능  
>
> #### 2. [엑셀](엑셀.md)  
> - 화면의 기수 목록을 **엑셀 파일로 다운로드**  
> - 단순조회, 내부 보고, 고객사 전달용 활용  
>
> #### 3. 선택 체크박스  
> - 기수 선택 가능  
> - 선택 후 **삭제, 엑셀 다운로드** 등 작업 수행  
>
> #### 4. 삭제  
> - 선택한 기수 삭제  
> - 삭제 전 확인 팝업 표시 → 삭제 후 복구 불가  
>
> #### 5. 기수명 클릭  
> - 기수명 클릭 시 **상세 정보** 하단에 표시  
> - 포함 과정, 강사, 수강 인원 등 세부 내용 확인·수정 가능  
>
> #### 6. 검색  
> - 기수명, 과정명 등 조건 입력 → 목록 검색  
> - 조건 초기화: 입력값 삭제 후 **Enter** 키  

---

> [!tip]- 사용방법  
> - **신규 등록**: `[신규등록]` 버튼 클릭 → 기수 정보 입력 → 저장  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 단순조회, 내부 보고, 고객사 전달용 저장  
> - **삭제**: 체크박스 선택 → `[삭제]` 클릭 → 확인 후 삭제  
> - **상세 확인**: 기수명 클릭 → 상세 정보 확인·수정  
> - **검색 활용**: 검색창에 기수명/과정명 입력 → Enter → 조건에 맞는 기수 조회
