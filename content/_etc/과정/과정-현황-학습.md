## 기능 설명  

> [!note]- #### 1. [엑셀](과정-현황-운영-엑셀.md)  
> - 현재 화면에 표시된 운영 현황 데이터를 **엑셀 파일로 다운로드**  
> - 내부 보고, 고객사 전달, 운영 현황 관리용으로 활용 가능  

> [!note]- #### 2. 체크박스  
> - 학습자별 행 선택 기능  
> - 다중 선택 후 **수료 / 강제수료 / 결제 미수료 / 취소 / 일자변경** 등 기능 실행 가능  

> [!note]- #### 3. 수료 처리  
> - 선택된 학습자의 상태를 **수료**로 변경  
> - 정상 학습 완료 조건(진도율, 시험, 과제 등) 충족 시 사용, 수료증 발급 대상 처리  

> [!note]- #### 4. 강제 수료 처리  
> - 조건 미충족 여부와 관계없이 **강제 수료 처리**  
> - 행정 처리, 예외 상황 등 특수 케이스 활용  

> [!note]- #### 5. 결제 미수료 처리  
> - 학습 완료했으나 결제 미완료 건을 **결제 미수료 상태**로 변경  
> - 수료증 발급 제한, 결제 완료 후 재처리 가능  

> [!note]- #### 6. 수강 취소  
> - 학습자의 과정 참여를 **취소 상태**로 변경 

> [!note]- #### 7. 일자 변경  
> - 과정 시작일/종료일 변경 가능  
> - 학습 일정 조정 시 활용 

> [!note]- #### 8. 이름 클릭  
> - 학습자 이름 클릭 시 **상세 페이지** 이동 
> - 기본 정보, 학습 이력, 진도율, 성취도 확인 가능

> [!note]- #### 9. 과정명 클릭  
> - 학습자가 수강 중인 과정의 **상세 페이지**로 이동  
> - 과정 개요, 운영 정보, 학습자별 진도 현황 확인 가능  

> [!note]- #### 10. 검색  
> - 이름, ID, 과정명, 기수 등 조건으로 학습 내역 검색 가능 
> - 조건별 빠른 조회 가능

---

> [!tip]  
> ## 사용방법  
> - **엑셀 다운로드**: [엑셀] 버튼 클릭 → 현재 학습 목록 저장  
> - **체크박스 활용**: 다수 학습자 선택 → 상태 변경/일자 조정  
> - **수료 / 강제수료 / 결제 미수료 / 취소**: 체크박스 선택 후 해당 버튼 클릭  
> - **일자 변경**: 체크박스 선택 → [일자변경] 버튼 클릭  
> - **상세 확인**: 학습자 이름 클릭 → 상세 페이지, 과정명 클릭 → 과정 상세 페이지  
> - **검색 활용**: 검색창에 조건 입력 → 조회 버튼 클릭 → 원하는 학습자 확인
