## 기능 설명  

> [!note]- #### 1. [엑셀](엑셀.md)  
> - 현재 화면에 표시된 운영 현황 데이터를 **엑셀 파일로 다운로드** 
> - 내부 보고용, 고객사 전달용, 운영 현황 관리용 활용 가능

> [!note]- #### 2. 과정명 클릭  
> - 과정명을 클릭하면 해당 **과정관리 페이지**로 이동
> - 상세 페이지에서 학습자 목록, 수료 처리, 성적 확인 등 추가 관리 기능 활용 가능 

> [!note]- #### 3. 검색  
> - 과정명, ID, 기수명 등 조건 입력 → 원하는 과정 빠르게 검색
> - 조건 검색으로 특정 과정만 선별 필터링 가능

---

> [!tip]  
> ## 사용방법  
> - **엑셀 다운로드**: [엑셀] 버튼 클릭 → 현재 수료 현황 목록 저장  
> - **상세 확인**: 과정명 클릭 → 학습자 목록, 수료 처리, 성적 확인 등 상세 정보 확인  
> - **검색 활용**: 검색창에 과정명/ID/기수명 입력 → 조회 클릭 → 조건에 맞는 수료 현황 확인
