## 기능 설명  

> [!note]- #### 1. 신규등록  
> - 새로운 **템플릿** 등록.  
> - 클릭 시 **템플릿 등록 화면**으로 이동.  

> [!note]- #### 2. 엑셀  
> - 현재 화면의 템플릿 목록 **엑셀 다운로드**.  
> - 단순조회, 내부 보고, 고객사 전달용 활용.  

> [!note]- #### 3. 체크박스  
> - 템플릿 선택 기능.  
> - 삭제 또는 일괄 작업 시 사용, 다중 선택 가능.  

> [!note]- #### 4. 삭제  
> - 체크박스 선택 템플릿 삭제.  
> - 삭제 전 확인 메시지 표시, 삭제 후 복구 불가.  

> [!note]- #### 5. 과정 수  
> - 템플릿에 연결된 과정 수 표시.  
> - 숫자 클릭 시 연결된 과정 목록 확인 가능.  

> [!note]- #### 6. 제목  
> - 템플릿 제목 표시.  
> - 제목 클릭 시 **템플릿 수정 화면**으로 이동.  

> [!note]- #### 7. 검색  
> - 제목, ID 등 조건으로 템플릿 검색.  
> - 원하는 템플릿 빠르게 조회 가능.  

---

> [!tip]  
> ## 사용방법  
> - **신규 등록**: `[신규등록]` 클릭 → 템플릿 정보 입력 → 저장.  
> - **엑셀 다운로드**: `[엑셀]` 클릭 → 현재 템플릿 목록 저장.  
> - **템플릿 선택**: 체크박스로 하나 이상 선택.  
> - **템플릿 삭제**: 선택 후 `[삭제]` 클릭 → 확인 메시지 후 삭제 완료.  
> - **연결 과정 확인**: 과정 수 숫자 클릭 → 연결 과정 목록 확인.  
> - **상세 확인**: 제목 클릭 → 템플릿 수정 화면 이동.  
> - **검색 활용**: 제목, ID 입력 후 조회 → 조건에 맞는 템플릿 확인.
