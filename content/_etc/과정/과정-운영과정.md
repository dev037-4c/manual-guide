> [!note]- 기능 설명  
> 1. [신규등록](과정-신규등록.md)  
> - 새로운 **운영과정** 등록  
> - 클릭 시 **운영과정 등록 화면**으로 이동  
>
> 2. 선택 체크박스  
> - 특정 과정 선택 기능  
> - **삭제 / 복사 / 엑셀 다운로드** 등 일괄 작업 시 활용 가능  
> - 다중 선택 지원  
>
> 3. 과정복사  
> - 기존 운영과정을 복사하여 새로운 과정 생성  
> - 동일 설정 과정 빠른 등록 가능  
>
> 4. 삭제  
> - 선택된 운영과정 삭제  
> - 삭제 전 확인 메시지 표시, 삭제 후 접근 불가  
>
> 5. [엑셀](엑셀.md)  
> - 현재 화면의 과정 목록 **엑셀 다운로드**  
> - 단순조회, 내부 보고, 고객사 전달용 활용  
>
> 6. 수료확정  
> - 학습 완료자의 **수료 여부 최종 확정**  
> - 확정 후 학습자는 **수료증 발급 대상** 처리  
>
> 7. 학습중  
> - 학습자 참여 과정 상태 확인 및 관리  
> - 진도율, 시험/과제 현황 실시간 확인 가능  
>
> 8. 학습중지  
> - 학습자 과정 참여 **중지 처리**  
> - 개인 사정 또는 운영자 판단에 따라 제한 가능  
>
> 9. 과정관리  
> - 선택 과정의 **상세 관리 화면** 이동  
> - 과정 정보 수정, 학습자 관리, 진도/평가/수료 현황 확인 가능  
>
> 10. 제목 클릭  
> - 과정명 클릭 → **과정 수정 화면** 이동  
> - 과정 개요, 운영 정보, 학습자 현황 확인 및 편집 가능  
>
> 11. 분류표시 토글  
> - 과정 목록을 **분류 기준(예: 상시제/기수제)**으로 표시/숨김 가능  
>
> 12. 검색  
> - 운영과정을 **제목, ID, 기수 등 조건**으로 검색  
> - 원하는 과정 빠르게 조회 가능  

---

> [!tip]- 사용방법  
> - **신규 등록**: `[신규등록]` 클릭 → 과정 정보 입력 → 저장  
> - **과정 선택**: 체크박스 선택 → `[삭제]`, `[복사]`, `[엑셀 다운로드]` 등 작업 수행  
> - **과정 복사**: `[과정복사]` 클릭 → 동일 설정 과정 빠르게 생성  
> - **과정 삭제**: 체크박스 선택 → `[삭제]` 클릭 → 확인 후 삭제 완료  
> - **엑셀 다운로드**: `[엑셀]` 클릭 → 현재 과정 목록 저장  
> - **수료 확정**: `[수료확정]` 클릭 → 학습 완료자 수료 여부 확정 → 수료증 발급 대상 처리  
> - **학습중 확인**: `[학습중]` 클릭 → 진도율, 시험, 과제 등 진행 현황 확인  
> - **학습 중지**: `[학습중지]` 클릭 → 학습자 과정 참여 중지 처리  
> - **과정 관리**: `[과정관리]` 클릭 → 상세 관리 화면(정보 수정, 학습자/진도/평가/수료 관리)  
> - **상세 확인**: 과정명 클릭 → 과정 수정 화면 이동  
> - **분류 표시/숨김**: 분류 토글 클릭 → 분류 기준별 표시 전환  
> - **검색 활용**: 제목, ID, 기수 입력 → 조건에 맞는 과정 조회
