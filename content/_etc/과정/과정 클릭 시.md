***

> [!note]- 기능 설명
>   
> #### 1. [목록](과정-패키지.md)  
> - 패키지 목록 화면으로 이동함  
> - 상세 화면에서 나와 전체 패키지 목록 확인 가능  
>
> #### 2. 체크박스  
> - 개별 과정 선택 가능  
> - 다중 선택 후 **저장, 삭제** 등에 활용  
>
> #### 3. 저장  
> - 선택·수정한 과정 정보 저장  
> - 패키지 구성 유지·갱신 시 사용  
>
> #### 4. 삭제  
> - 선택한 과정 패키지에서 제거  
> - 삭제 시 목록에서 제외됨  
>
> #### 5. 추가  
> - 새로운 과정 패키지에 등록  
> - 기존 패키지 보강·수정 시 사용  
>
> #### 6. 과정 제목 클릭  
> - 제목 클릭 시 **상세/수정 페이지**로 이동  
> - 과정 개요, 운영 정보, 학습 현황 확인·수정 가능  

---

>[!tip]- 사용방법
> - **목록 이동**: [목록] 버튼 클릭 → 패키지 목록으로 이동  
> - **체크박스 활용**: 과정 선택 후 저장·삭제 실행  
> - **과정 저장**: 선택·수정한 과정 정보 저장  
> - **과정 삭제**: 선택한 과정 패키지에서 제거  
> - **과정 추가**: `[추가]` 버튼 클릭 → 새 과정 등록  
> - **상세 확인**: 과정 제목 클릭 → 상세/수정 페이지로 이동
