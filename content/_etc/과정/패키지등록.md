> [!note]- 기능 설명  
> #### 1. 목록
> - 현재 작성 중인 패키지 입력을 종료하고 목록 화면으로 이동한다.
> - 이동 시 입력 값은 저장되지 않으므로, 이동 전 저장이 필요하다.
>
> #### 2. 저장
> - 입력한 패키지 기본 정보(제목, 대상, 가격, 기간 등)를 저장한다.
> - 신규 등록 및 수정 사항이 모두 반영된다.
>
> #### 3. 삭제
> - 현재 등록 중인 패키지를 삭제한다.
> - 삭제 후에는 복구가 불가능하므로 주의가 필요하다.
> 
> #### 4. 과정
> - 패키지에 포함할 개별 과정을 관리할 수 있는 과정 선택 화면으로 이동한다.
> 
> #### 5. 파일선택
> - 패키지 대표 이미지를 업로드한다.
> - 등록된 이미지는 썸네일로 표시되어 학습자가 확인할 수 있다.
> 
>  #### 6. 기간동일
>  - 신청기간과 취소기간을 동일한 날짜로 자동 설정한다.
>  - 동일 기간 적용 시 편리하게 활용된다.

---

> [!tip]- 사용방법  
> - **목록**: 등록된 패키지를 확인하려면 **목록 버튼 ①** 클릭 → 패키지 목록으로 이동한다.
> - **저장**: 입력한 정보를 반영하려면 **저장 버튼 ②** 클릭 → 신규 등록 또는 수정 내용을 저장된다.
> - **삭제**: 불필요한 패키지를 제거하려면 **삭제 버튼 ③** 클릭 → 확인 창에서 승인하면 삭제된다.
> - **과정**: 패키지에 강좌를 포함하려면 **과정 버튼 ④** 클릭 → 과정 선택 및 추가를 진행한다.
> - **대표 이미지**: 학습자 화면에 표시될 이미지를 지정하려면 **파일선택 버튼 ⑤** 클릭 → 이미지가 업로드된다.
> - **기간 동일**: 신청기간과 취소기간을 같은 날짜로 맞추려면 **기간동일 버튼 ⑥** 클릭 → 두 기간이 자동으로 일치된다.
