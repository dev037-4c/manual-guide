> [!note]- 기능 설명  
> 1. 기간 필터  
> - `금년`, `최근 1년`, `최근 6개월`, `최근 3개월` 중 하나를 선택하여 조회 기간을 빠르게 설정  
> - 클릭 시 해당 기간의 검색 키워드 데이터가 자동 반영  
> 2. 기간 직접 설정  
> - 달력 아이콘을 통해 **조회 시작일과 종료일**을 직접 지정  
> - 특정 기간의 데이터를 확인하고 싶을 때 사용  
> 3. 상세 데이터  
> - 그래프의 막대 위에 마우스를 올리면 **해당 키워드의 검색 횟수**가 팝업으로 표시  
> - 시각적으로 데이터 비교 시 유용  
> 4. 엑셀  
> - 현재 조회된 목록을 엑셀 파일로 다운로드  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용  
> 5. 기간 필터 (목록)  
> - 목록 화면에서도 기간을 직접 설정  
> - 설정된 조건에 맞는 검색어 목록만 표시  
> 6. 검색어 컬럼  
> - 목록 형태로 검색어별 **검색 횟수**를 제공  
> - 컬럼 정렬을 통해 많이 검색된 키워드 순으로 빠르게 확인 가능  

---

> [!tip]- 사용방법  
> - **기간 선택**: `[금년]`, `[최근 1년]`, `[최근 6개월]`, `[최근 3개월]` 버튼 클릭  
> - **기간 지정**: 시작일–종료일 선택 후 `[조회]` 클릭  
> - **그래프 활용**: 막대에 마우스 오버 → 키워드별 검색 횟수 확인  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 현재 목록을 저장  
> - **목록 확인**: 검색어/검색횟수 테이블에서 세부 데이터 확인
