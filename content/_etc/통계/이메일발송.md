## 기능 설명

> [!note]- ### 1. [엑셀](엑셀.md)  
> - 현재 조회된 목록을 엑셀 파일로 다운로드합니다.  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용할 수 있습니다.

> [!note]- ### 2. 내용 확인  
> - 내용 수를 클릭하면 상세 내용을 확인 할수있는 창에서 확인 및 관리를 할 수 있다 


> [!note]- ### 3. 검색  
> - 조건 검색을 통해 원하는 발송 내역을 조회할 수 있습니다.  
> - 검색 가능한 주요 조건:  
>   - 이름, ID  
>   - IP  
>   - 목적  
>   - 제목, 내용  
>   - 발송일

> [!tip]
> ## 사용방법  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 현재 조회된 결과를 엑셀 파일로 저장  
> - **내용 확인**: 내용 수 클릭 → 해당 항목의 상세 내용 확인 및 관리 가능  
> - **검색 실행**: 검색창에 조건 입력 후 `[검색]` 클릭 → 조건에 맞는 이메일 발송 내역 조회
