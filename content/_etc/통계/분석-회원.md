## 기능 설명

> [!note]- #### 1. 빠른 기간 선택  
> - 분석 기간을 빠르게 지정할 수 있는 기능
> - 옵션: `금년`, `최근 1년`, `최근 6개월`, `최근 3개월`  
> - 선택 시 해당 기간 동안의 **누적회원 / 회원등록** 수가 차트에 표시

> [!note]- #### 2. 기간 직접 설정  
> - 시작일과 종료일을 직접 입력하여 원하는 기간의 데이터를 조회
> - `[조회]` 버튼 클릭 시, 해당 기간의 **회원 증가 현황**이 차트에 반영

> [!note]- #### 3. 상세 데이터  
> - 그래프의 막대 또는 라인 위에 마우스를 올리면, 해당 월의 **상세 데이터(누적회원/회원등록 수)** 가 툴팁으로 표시

> [!tip]
> ## 사용방법  
> - **빠른 조회**: `[금년]`, `[최근 1년]`, `[최근 6개월]`, `[최근 3개월]` 버튼 클릭 → 차트 데이터 자동 갱신  
> - **세부 조회**: 시작일/종료일을 직접 입력한 후 `[조회]` 클릭 → 맞춤 기간의 회원 현황 확인  
> - **상세 확인**: 그래프 위 마우스 오버 → 해당 월의 누적회원/회원등록 수 확인**
