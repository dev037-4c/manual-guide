## 기능 설명

> [!note]- #### 1. [엑셀](엑셀.md)  
> - 현재 조회된 목록을 엑셀 파일로 다운로드
> - 단순 조회, 내부 보고, 고객사 전달용 자료로 활용

> [!note]- #### 2. 개별 선택 체크박스  
> - 특정 학습자를 선택
> - 선택된 학습자에 대해 일괄 처리(예: 수료 처리 취소)가 가능

> [!note]- #### 3. 수료처리취소  
> - 선택한 학습자의 수료 처리를 **취소**하는 기능
> - 잘못된 수료 처리 건을 되돌리거나, 조건 미충족 시 재처리할 때 사용

> [!note]- #### 4. 과정명 클릭  
> - 과정명을 클릭하면 **해당 과정의 상세 페이지**로 이동
> - 상세 페이지에서 과정 정보, 참여 학습자, 수료 조건 등을 확인

> [!note]- #### 5. 검색  
> - 검색 토글을 활성화하여 다양한 조건으로 조회
> - 이름, ID, 기수, 과정명, 수료증 발급 여부 등으로 검색이 가능

> [!tip]
> ## 사용방법  
> - **엑셀 다운로드**: `[엑셀]` 버튼 클릭 → 현재 조회 결과 저장  
> - **개별 선택**: 체크박스 선택 → 선택된 학습자에 대해 일괄 처리 가능  
> - **수료처리 취소**: 체크박스 선택 후 `[수료처리취소]` 버튼 클릭  
> - **과정 상세 확인**: 과정명 클릭 → 상세 페이지로 이동  
> - **검색 실행**: 검색 토글 활성화 → 조건 입력 후 조회 실행
