## 기능 설명

> [!note]- #### 1. 페이지 선택 체크박스  
> - 분석 대상 페이지를 선택
> - 예: `사용자메인페이지만` 체크 → 사용자 메인 화면의 평균 체류시간만 표시

> [!note]- #### 2. 빠른 기간 선택  
> - 기간별 빠른 조회 기능을 제공
> - 옵션: `금년`, `최근 1년`, `최근 6개월`, `최근 3개월`  
> - 클릭 시 선택한 기간의 페이지별 그래프가 표시

> [!note]- #### 3. 기간 직접 설정  
> - 시작일과 종료일을 직접 지정
> - [조회] 버튼 클릭 시 해당 기간의 데이터가 차트에 표시

> [!note]- #### 4. 상세 데이터  
> - 특정 그래프 영역에 마우스를 올리면 상세 수치를 확인

> [!tip]
> ## 사용방법  
> - **페이지 선택**: 체크박스를 선택해 특정 페이지 그래프 확인  
> - **빠른 조회**: `[금년]`, `[최근 1년]`, `[최근 6개월]`, `[최근 3개월]` 클릭 → 해당 기간의 데이터 표시  
> - **세부 조회**: 시작일과 종료일을 직접 설정 후 `[조회]` 클릭 → 맞춤 기간 데이터 확인  
> - **상세 확인**: 그래프에 마우스 오버 → 평균 체류시간(초) 확인
