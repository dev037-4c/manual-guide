## 기능 설명

> [!note]- #### 1. [엑셀](공통기능-엑셀.md)  
> - 현재 조회된 목록을 엑셀 파일로 다운로드합니다.  
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용할 수 있습니다.

> [!note]- #### 2. 날짜 선택  
> - 로그인 일시를 기준으로 **특정 일자를 선택**할 수 있습니다.  
> - 캘린더 팝업에서 원하는 날짜를 클릭하면 해당 일자의 로그인/세션 이력이 표시됩니다.

> [!note]- #### 3. 사이트 구분 필터  
> - 조회 대상을 **관리자 / 사용자**로 구분하여 필터링할 수 있습니다.  
> - 예: `관리자` 선택 시 관리자 계정의 접속 이력만 표시됩니다.

> [!note]- #### 4. 세션 ID  
> - 특정 **세션 ID**를 클릭하면 해당 세션의 활동 내역을 직접 조회할 수 있습니다.  
> - 동일 사용자의 여러 접속 세션 중 특정 세션만 추적할 때 유용합니다.

> [!tip]
> ## 사용방법  
> - **엑셀 저장**: `[엑셀]` 버튼 클릭 → 현재 화면의 데이터를 다운로드  
> - **기간 조회**: 캘린더에서 날짜 선택 → 해당 일자 기준 로그인/세션 정보 확인  
> - **대상 선택**: 사이트 구분 필터(관리자 / 사용자) 적용 → 계정 유형별 접속 이력 조회  
> - **세션 추적**: 세션 ID 입력 후 검색 → 해당 세션의 상세 이력 확인
