## 기능 설명

> [!note]- #### 1. 페이지 선택 체크박스  
> - 분석 대상 페이지를 선택
> - 예: `사용자메인페이지` 체크 시 해당 페이지의 세션 수만 표시
> - 복수 선택이 가능하여 여러 페이지를 동시에 비교

> [!note]- #### 2. 빠른 기간 선택  
> - 최근 단위 기간별 조회 기능을 제공
> - 옵션: `최근 6개월`, `최근 3개월`, `최근 2개월`, `최근 1개월`  
> - 클릭 시 해당 기간의 **일별 세션 데이터**가 차트에 표시

> [!note]- #### 3. 기간 직접 설정  
> - 시작일과 종료일을 직접 지정
> - [조회] 버튼 클릭 시 선택한 기간의 **일별 세션 수**가 표시

> [!note]- #### 4. 상세 데이터  
> - 그래프 위 특정 날짜에 마우스를 올리면 해당 일자의 **세션 수**를 확인

> [!tip]
> ## 사용방법  
> - **페이지 선택**: 원하는 페이지 유형 체크박스 선택 → 해당 페이지의 세션만 표시  
> - **빠른 조회**: `[최근 6개월]`, `[최근 3개월]`, `[최근 2개월]`, `[최근 1개월]` 클릭  
> - **세부 조회**: 시작일/종료일 직접 선택 후 `[조회]` 클릭  
> - **상세 확인**: 그래프 위 마우스를 올리면 → 해당 일자의 세션 수 팝업 표시
