## 기능 설명

> [!note]- #### 1. 페이지 선택 체크박스  
> - 분석할 대상 페이지를 선택
> - 예: `사용자메인페이지` 체크 시 해당 페이지의 월별 순방문만 표시

> [!note]- #### 2. 빠른 기간 선택  
> - 최근 단위 기간별 조회 기능을 제공
> - 옵션: `금년`, `최근 1년`, `최근 6개월`, `최근 3개월`  
> - 클릭 시 선택한 기간의 **월별 순방문 데이터**가 표시

> [!note]- #### 3. 기간 직접 설정  
> - 시작월과 종료월을 직접 지정
> - [조회] 버튼 클릭 시 해당 기간의 **월별 순방문 수**가 표시

> [!note]- #### 4. 상세 데이터  
> - 그래프의 특정 월에 마우스를 올리면 해당 월의 **순방문 수**를 확인

> [!tip]
> ## 사용방법  
> - **페이지 선택**: 원하는 페이지 유형 체크박스 선택 → 해당 페이지의 월별 순방문만 표시  
> - **빠른 조회**: `[금년]`, `[최근 1년]`, `[최근 6개월]`, `[최근 3개월]` 클릭  
> - **세부 조회**: 시작월/종료월 직접 선택 후 `[조회]` 클릭  
> - **상세 확인**: 그래프 위 마우스 오버 → 특정 월의 순방문 수 확인
