## 콘텐츠 관리_리소스

등록된 학습 콘텐츠(WBI, SCORM, 유튜브 등)를 확인하고 관리하는 화면입니다.  
신규 콘텐츠 등록부터 삭제, 미리보기까지 모든 기능을 통합 제공하는 핵심 관리 메뉴입니다.

***

![[학습-콘텐츠.png]]

***

> **[콘텐츠] → [학습] → [콘텐츠]** 메뉴를 클릭하여 이동한 페이지

***

## 기능 설명

#### 1. 콘텐츠 유형 탭

- 탭: [리소스](학습-콘텐츠.md), [WBI](WBI.md), [SCORM](scorm.md), [리소스 그룹](리소스그룹.md)
- 클릭 시 해당 유형의 콘텐츠만 필터링
  
***

#### 2.  [신규등록](학습-콘텐츠-신규등록.md)

- 신규등록 버튼 클릭 시 콘텐츠 신규등록 페이지로 이동
- 새로운 콘텐츠를 추가할 때 사용
  
***
  
#### 3. [엑셀](엑셀.md)

- 엑셀 버튼 클릭 시 엑셀 다운로드 팝업 창 생성
- 콘텐츠 목록을 엑셀 파일로 저장 가능
  
***

#### 4. 삭제

- 삭제 버튼 클릭 시 해당 콘텐츠 삭제 (체크박스 선택 시 활성화)
- 선택한 콘텐츠 삭제

***

#### 5. 체크박스

- 개별 또는 다중 콘텐츠 선택 가능
- 삭제, 일괄 작업 시 활용

***
  
#### 6. [상세](학습-콘텐츠-상세.md)

- ID/제목 클릭 시 콘텐츠 상세 페이지로 이동

***

#### 7. [미리보기](학습-콘텐츠_미리보기.md)

- 미리보기 버튼 클릭 시 콘텐츠(예: 동영상)를 바로 확인 가능

***

#### 8. 유형필터

- 콘텐츠 유형(동영상, 오디오, PDF, HTML, 비메오, 유튜브, 웹에디터)별로 필터링 가능

***

#### 9. 검색

- ID, 제목 등 원하는 항목을 검색할 수 있는 기능

***

## 사용방법

- 특정 콘텐츠 유형 페이지로 이동하려면 상단의 **옵션(리소스, WBI, SCORM, 리소스 그룹)** 을 선택합니다.
- 새로운 콘텐츠를 등록하려면 **신규등록 버튼**을 클릭하여 신규등록 페이지에서 정보를 입력합니다.
- 콘텐츠 목록을 저장하려면 **엑셀 버튼**을 클릭하여 팝업에서 다운로드를 진행합니다.
- 전체 콘텐츠 목록을 다운로드하려면 **다운로드 버튼**을 클릭합니다.
- 콘텐츠를 삭제하려면 삭제할 항목을 **체크박스로 선택 후 삭제 버튼**을 클릭합니다.
- 특정 콘텐츠의 상세 정보를 확인하려면 **ID 또는 제목**을 클릭합니다.
- 콘텐츠를 미리 확인하려면 해당 항목의 **미리보기 버튼**을 클릭합니다.
- 원하는 유형별 콘텐츠만 보고 싶다면 **유형 필터(동영상, 오디오, PDF 등)** 를 선택합니다.
- 특정 콘텐츠를 찾으려면 **검색창에 ID 또는 제목**을 입력하여 조회합니다.
