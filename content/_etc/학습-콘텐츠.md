# 9. 콘텐츠

## [[학습-콘텐츠]]

---



---

> **[콘텐츠] → [학습] → [콘텐츠]** 메뉴를 클릭하여 이동한 페이지

---

### 기능 설명

1. **각 옵션 선택** 시 해당 페이지로 이동 (리소스, WBI, SCORM, 리소스 그룹)  
2. **신규등록 버튼** 클릭 시 콘텐츠 [신규등록](학습-콘텐츠-신규등록.md) 페이지로 이동  
3. **엑셀 버튼** 클릭 시 엑셀 다운로드 팝업 창 생성, 콘텐츠 목록을 엑셀 파일로 저장  
4. **다운로드 버튼** 클릭 시 전체 콘텐츠 목록 다운로드  
5. **삭제 버튼** 클릭 시 해당 콘텐츠 삭제 (체크박스 선택 시 활성화)  
6. 콘텐츠 **체크박스 선택** 가능  
7. **ID/제목 클릭** 시 콘텐츠 [상세](학습-콘텐츠-상세.md) 페이지로 이동  
8. **미리보기 버튼** 클릭 시 동영상 미리보기 가능  
9. 콘텐츠 **유형 선택 및 필터링** 가능 (동영상, 오디오, PDF, HTML, 비메오, 유튜브, 웹에디터) 
10. **검색 기능**을 통해 ID, 제목 등 원하는 항목 검색 가능
