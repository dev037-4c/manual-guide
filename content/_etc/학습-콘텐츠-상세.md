---

### 기능 설명

1. **목록 버튼** 클릭 시 [학습 - 콘텐츠](학습-콘텐츠.md) 목록 페이지로 이동  
2. **저장 버튼** 클릭 시 입력된 콘텐츠 정보 저장  
3. **복사 버튼** 클릭 시 해당 콘텐츠 복사하여 신규 등록 가능  
4. **미리보기 버튼** 클릭 시 영상 미리보기 창 열림  
5. **설정 버튼** 클릭 시 플레이어 옵션, 자막, 영상화면, 전체화면 설정 가능
6. **시간 가져오기 버튼** 클릭 시 영상 재생 위치의 시간을 자동으로 입력  
7. **마커 추가 버튼** 클릭 시 입력된 시간·제목·내용을 마커 목록에 등록  
8. **마커 삭제 버튼** 클릭 시 선택된 마커를 목록에서 삭제
