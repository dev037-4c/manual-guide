## 출제그룹 관리

출제 문항을 분류하고 관리할 수 있도록 그룹화하는 기능 페이지입니다.  
각 그룹은 과목/주제 단위로 설정하여 시험 문제와 연결됩니다.

***

![[시험-출제그룹.png]]

***

> **[콘텐츠] → [시험] → [출제그룹]** 을 클릭하여 이동한 페이지  

***

## 기능 설명

#### 1. [신규등록](시험-출제그룹-신규등록.md)

- 클릭 시 출제그룹 신규등록 페이지로 이동

---

#### 2. [엑셀](엑셀.md)

- 클릭 시 엑셀 다운로드 팝업 창 생성
- 다운로드 버튼 클릭 시 설정한 사유에 맞는 엑셀 파일 다운로드 가능

---

#### 3. 삭제

- 체크박스로 항목 선택 후 삭제 버튼 클릭 시 해당 그룹 삭제

---

#### 4. 체크박스

- 개별 출제그룹 선택 가능
- 선택된 그룹은 삭제 가능

---

#### 5. 제목

- 제목 클릭 시 해당 일반시험 상세정보 페이지로 이동

---

#### 6. 검색

- 원하는 검색어 입력 시 검색어가 포함된 항목 표시

---

## 사용방법

- 새로운 출제그룹을 만들려면 **신규등록 버튼**을 클릭하여 등록 페이지로 이동합니다.
- 현재 출제그룹 목록을 저장하려면 **엑셀 버튼**을 클릭하고, 팝업에서 **다운로드 버튼**을 눌러 파일을 내려받습니다.
- 필요 없는 그룹을 삭제하려면 **체크박스로 그룹을 선택**한 후 **삭제 버튼**을 클릭합니다.
- 특정 시험 그룹의 상세 내용을 확인하려면 해당 그룹의 **제목을 클릭**합니다.
- 원하는 그룹을 빠르게 찾으려면 우측 상단의 **검색창**에 검색어를 입력합니다.
