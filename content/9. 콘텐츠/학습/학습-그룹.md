# 9. 콘텐츠

## [[학습-그룹]]

---

![[학습-그룹.png]]

---

> **[콘텐츠] → [학습] → [그룹]** 메뉴를 클릭하여 이동한 페이지

---

### 기능 설명

1. **신규등록 버튼** 클릭 시 그룹 신규등록 페이지로 이동  
2. **엑셀 버튼** 클릭 시 엑셀 다운로드 팝업 창 생성  
3. **엑셀 다운로드 팝업**에서 보고서 유형 선택 후 다운로드 가능
4. **삭제 버튼** 클릭 시 해당 그룹 삭제 (체크박스 선택 시 활성화)  
5. 그룹 **체크박스 선택** 가능  
6. **ID/제목 클릭** 시 그룹 상세 페이지로 이동  
7. **검색 기능**을 통해 ID, 제목 등 원하는 항목 검색 가능
