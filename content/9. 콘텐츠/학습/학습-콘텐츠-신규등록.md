# 9. 콘텐츠

---

![[학습-콘텐츠-신규등록.png]]

---

> **[콘텐츠] → [학습] → [콘텐츠] → [신규등록] 클릭하여 이동한 페이지

---

### 기능 설명

1. **목록 버튼** 클릭 시 [학습] - [콘텐츠] 페이지로 이동  
2. **저장 버튼** 클릭 시 입력된 콘텐츠의 정보 저장  
3. 원하는 **콘텐츠/파일 유형** 선택  
4. **파일선택 버튼** 클릭 시 파일 선택 창 생성  
5. **자막 파일 등록** 또는 **AI 자막 생성**  
6. **가져오기** 버튼 클릭 시 재생시간 확인 가능
7. **플레이어 설정** 기능 선택  
8. 운영권한 선택 가능
