# 9. 콘텐츠

## [[시험-출제그룹]]

---

![[시험-출제그룹.png]]

---

> **[콘텐츠] → [시험] → [출제그룹]** 을 클릭하여 이동한 페이지

---

### 기능 설명

1. **신규등록 버튼** 클릭 시 출제그룹 신규등록 페이지로 이동  
2. **엑셀 버튼** 클릭 시 엑셀 다운로드 팝업 창 생성  
3. **다운로드 버튼** 클릭 시 설정한 사유에 맞는 엑셀 파일 다운로드  
4. 출제그룹 **체크박스 선택** 가능  
5. **제목** 클릭 시 해당 일반시험 상세정보 페이지로 이동
6. 원하는 검색어 입력 시 검색어가 포함된 항목 표시
