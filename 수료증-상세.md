> [!note]- 기능설명  
> #### 1. 언어 선택  
> - 한국어, 영어, 중국어, 일본어 중 선택 가능하다.  
> - 수료증 내용을 다국어 버전으로 관리할 때 활용된다.  
>
> #### 2. 목록  
> - 수료증 등록/수정 화면에서 목록 화면으로 돌아갈 수 있다.  
> - 전체 수료증 현황을 다시 확인할 때 사용된다.  
>
> #### 3. 저장  
> - 입력한 수료증 정보를 저장한다.  
> - 제목, 인증서 이름, 직인 이미지 등의 정보가 반영된다.  
>
> #### 4. 삭제  
> - 선택된 수료증을 목록에서 삭제한다.  
> - 삭제된 항목은 복구할 수 없으므로 주의가 필요하다.  
>
> #### 5. 직인 이미지 업로드  
> - 기관 직인 이미지를 업로드하여 수료증에 표시한다.  
> - PNG, JPG 등 이미지 파일 형식을 지원한다.  

---

> [!tip]- 사용방법  
> 1. **언어 선택**: **언어 탭 ①** 클릭 → 한국어, 영어, 중국어, 일본어 중 원하는 언어를 선택한다.  
> 2. **목록**: **목록 버튼 ②** 클릭 → 수료증 목록 화면으로 이동한다.  
> 3. **저장**: 수료증 정보를 입력 후 **저장 버튼 ③** 클릭 → 입력한 내용이 저장된다.  
> 4. **삭제**: 수정 중이거나 선택된 수료증을 삭제하려면 **삭제 버튼 ④** 클릭 → 목록에서 제거된다.  
> 5. **직인 이미지 업로드**: **파일선택 버튼 ⑤** 클릭 → 기관 직인 이미지를 업로드해 수료증에 반영한다.
