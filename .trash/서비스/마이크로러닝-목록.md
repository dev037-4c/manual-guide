> [!note]- 기능 설명  
> #### 1. [신규등록](서비스-마이크로닝-목록-신규등록.md)
> - 새로운 마이크로러닝 과정을 생성한다.
> - 제목, 분류, 운영설정 등을 입력해 목록에 추가한다.
>
> #### 2. 체크박스
> - 그룹 목록에서 여러 항목을 동시에 선택할 수 있다.
> - 삭제와 같은 일괄 작업을 수행할 때 사용된다.
> 
> #### 3. 삭제
> - 선택된 마이크로러닝을 목록에서 삭제할 수 있다.
> - 체크박스로 선택한 항목만 삭제된다.
> 
> #### 4. [엑셀](엑셀.md)
> - 현재 화면에 표시된 마이크로러닝 목록을 엑셀 파일로 내려받을 수 있다.
> - 단순 조회, 내부 보고, 고객사 전달용으로 활용된다.
> 
> #### 5. 학습중
> - 선택한 과정을 학습중(운영 중) 상태로 전환한다.
> - 학습자에게 노출되어 수강 신청 및 학습 진행이 가능해진다.
> 
> #### 6. 학습중지
> - 선택한 과정을 학습중지 상태로 전환한다.
> - 일시적으로 수강 신청·학습을 제한할 때 사용한다.
> 
> #### 7. [과정관리](서비스-마이크로닝-목록-과정관리버튼클릭.md)
> - 주차추가: 과정 내 주차(차시)를 빠르게 생성한다. 템플릿/개수 지정으로 일괄 추가가 가능하다.
> - 순서변경: 과정 카드의 표시 순서/학습 진행 순서를 재정렬한다.
> 
> #### 8. 검색
> - 검색창에서 특정 키워드(ID, 제목 등)를 입력해 원하는 목록을 찾을 수 있다.
> 
> #### 9. [ID / 제목](서비스-마이크로닝-목록-ID_제목클릭.md)
> - 목록의 ID / 제목 영역을 클릭하면 상세수정 페이지로 이동한다.
> - 과정 기본정보, 공개/운영 설정, 주차·차시 구성, 이수조건 등을 편집한다.

---

> [!tip]- 사용방법  
> - **신규등록**: **신규등록 버튼 ①**을 클릭 → 제목·분류·운영설정을 입력 → **저장**하여 목록에 추가한다.
> - **체크박스**: 작업 대상 행의 **체크박스 ②**를 선택한다. 선택된 항목은 일괄 작업 대상이 된다.
> - **삭제**: 대상 선택 후 **삭제 버튼 ③**을 클릭 → 확인 창에서 **확인**을 눌러 삭제를 완료한다.
> - **엑셀 다운로드**: **엑셀 버튼 ④**를 클릭 → 저장 위치를 선택하면 현재 화면 구성대로 파일이 저장된다.
> - **학습중**: 대상 선택 후 **학습중 버튼 ⑤**를 클릭하면 상태가 **학습중**으로 변경된다.
> - **학습중지·주차추가**: 대상 선택 후 **학습중지 버튼 ⑥**을 클릭해 상태를 **학습중지**로 변경한다 → 이어서 **버튼 ⑦**을 클릭해 **주차추가**를 선택하고 추가할 **주차 수/템플릿**을 지정해 **저장**한다.
> - **순서변경**: **버튼 ⑦**을 클릭 → **순서변경**을 선택 → 목록을 **드래그** 또는 **번호/화살표**로 재정렬 → **적용/저장**한다.
> -**검색**: 원하는 발표를 찾으려면 **검색창 ⑧**에 조건 입력 → 결과를 조회한다.
> - **상세**: **제목/이미지 영역 ⑨**을 클릭 → **상세수정 페이지**에서 기본정보·운영 설정·주차/차시·이수조건을 수정 → **저장**한다.
